# [BJ_3035](https://acmicpc.net/problem/3035)

Print magnified picture with r, c factor

```txt
Input:
3 3 2 1
.x.
x.x
.x.

Output:
..xx..
xx..xx
..xx..
```

## Solution

```py
N, M, a, b = map(int, input().split())
G = [input() for _ in range(N)]
for i in range(N):
  for _ in range(a):
    for j in range(M):
      for _ in range(b):
        print(G[i][j], end='')
    print()
```
