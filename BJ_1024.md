* Given N and L, print consecutive L positive integers which sum is greater than N

```txt
Input: 18 2
Output: 5 6 7
```

{% endtab %}{% tab title='BJ_1024.py' %}

```py
n, l = map(int, input().split())
for i in range(l, 101):
  sm = n - i * (i - 1) // 2
  if sm % i == 0 and sm >= 0:
    print(*range(sm // i, sm // i + i))
    break
else:
  print(-1)
```
