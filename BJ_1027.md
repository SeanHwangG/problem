* Given list of building heights, find the maximum number of visible building from one building

```txt
Input:
15
1 5 3 2 6 3 2 6 4 2 5 7 3 1 5

Output: 7
```

{% endtab %}{% tab title='BJ_11758.py' %}

```py
N = int(input())
A = list(map(int, input().split(' ')))
B = [0] * N
for i in range(N):
  e = -99999999999
  for j in range(i + 1, N):
    d = (A[j] - A[i]) / (j - i)
    if d > e:
      e = d
      B[i] += 1
      B[j] += 1
res = 0
for i in B:
  res = max(res, i)
print(res)
```
