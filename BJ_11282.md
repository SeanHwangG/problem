* When N is given as an input, write a program to obtain the Nth korean letter

```txt
Input: 11172
Output: 힣
```

{% endtab %}{% tab title='BJ_11282.cpp' %}

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int n;
  cin >> n;
  wstring_convert<codecvt_utf8<char32_t>, char32_t> convert;
  cout << convert.to_bytes(0xAC00 + n - 1) << endl;
}
```
