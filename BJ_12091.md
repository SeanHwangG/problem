* Given cycle print next monster, Vaporeon->Jolteon->Flareon->

```txt
Input: Vaporeon
Output: Jolteon
```

{% endtab %}{% tab title='BJ_12091.cpp' %}

```cpp
#include<bits/stdc++.h>
int a;
std::string s[3] = {"Jolteon","Flareon","Vaporeon"};
int main(){
  std::cin >> a;
  std::cout << s[a % 3];
}
```

{% endtab %}{% tab title='BJ_12091.py' %}

```py
print(['Vaporeon','Jolteon','Flareon'][(1 + int(input())) % 3])
```
