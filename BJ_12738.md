* When given sequence A, find the longest increasing partial sequence

```txt
Input:
6
10 20 10 30 20 50

Output: 4
```

{% endtab %}{% tab title='BJ_12738.py' %}

```py
# O(N log N)
import bisect
input()
D = []
for e in map(int, input().split()):
  if len(D) == 0 or e > D[-1]:
    D += [e]
  else:
    D[bisect.bisect_left(D, e)] = e
print(len(D))
```
