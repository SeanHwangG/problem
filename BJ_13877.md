* Interpret given number in octa, decimal, hexa
* If inputted string cannot be interpreted as an octagonal number, print zero

```txt
Input:
4
1 1234
2 9
3 1777
4 129

Output:
1 668 1234 4660
2 0 9 9
3 1023 1777 6007
4 0 129 297
```

{% endtab %}{% tab title='BJ_13877.py' %}

```py
for _ in range(int(input())):
  a, b = input().split()
  print(a, end = ' ')
  if '8' in b or '9' in b:
    print(0, end = ' ')
  else:
    print(int(b, 8), end = ' ')
  print(int(b, 10), int(b, 16))
```
