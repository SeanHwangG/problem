* Print if equation is correct

```txt
Input: 2 + 2 = 5
Output: NO
```

{% endtab %}{% tab title='BJ_13985.go' %}

```go
package main
import "fmt"
func main() {
  var a, b, c int
  fmt.Scanf("%d + %d = %d", &a, &b, &c)
  if a + b == c {
    fmt.Printf("YES\n")
  }
  else {
    fmt.Printf("NO\n")
  }
}
```
