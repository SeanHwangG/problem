* When given a, b, c, find the largest circumference of triangle you can make

```txt
Input: 1 2 3
Output: 5
```

{% endtab %}{% tab title='BJ_14215.py' %}

```py
a, b, c = map(int, input().split())
print(min(a + b + c, (a + b + c - max(a, b, c)) * 2 - 1))
```
