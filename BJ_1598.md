* Given number, find manhattan difference
  * ![BJ_1598](images/20210227_034432.png)

```txt
Input: 11 33
Output: 8
```

{% endtab %}{% tab title='BJ_1598.py' %}

```py
a, b = map(int, input().split())
a -= 1
b -= 1
print(abs(a // 4 - b // 4) + abs(a % 4 - b % 4))
```
