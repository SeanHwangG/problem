* Given an arbitrary 3 multiple natural number of n
* output the number of ways to separate that number into 3 multiple natural numbers of 3

```txt
Input: 12
Output: 3
```

{% endtab %}{% tab title='BJ_16561.py' %}

```py
n = int(input())
res = n // 3 - 1
print(res * (res - 1) // 2)
```
