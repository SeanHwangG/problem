* Given c, k, p, print sum of over (1 ... c) Kn+Pn^2

```txt
Input: 3 1 1
Output: 20  # 2 6 12
```

{% endtab %}{% tab title='BJ_16673.py' %}

```py
c, k, p = map(int,input().split())
print(c * (c + 1) * k // 2 + c * (c + 1) * (2 * c + 1) * p // 6)
```
