* A Fixed charge, B won for each product, sells at C
* Given A, B, C, print break-even point

```txt
Input: 1000 70 170
Output: 11
```

{% endtab %}{% tab title='BJ_1712.py' %}

```py
A, B, C = map(int, input().split())
print(-1 if C <= B else A // (C - B) + 1)
```
