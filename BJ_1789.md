* Sum of distinct integers are S, find the maximum length

```txt
Input: 200
Output: 19   # 1 2 3 4 ... 18 29
```

{% endtab %}{% tab title='BJ_1789.py' %}

```py
s = int(input())
n = 1
while n * (n + 1) / 2 <= s:
  n += 1
print(n - 1)
```
