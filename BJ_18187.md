* How many plane can you create with n line?

```txt
Input: 3
Output: 7
```

{% endtab %}{% tab title='BJ_18187.py' %}

```py
count = 0
result = 1
for i in range(int(input())):
  if i == 0 or i % 3 != 0:
    count += 1
  result += count
print(result)
```
