* Given a, b, c, d, e, f, find x, y that solves following equation

$$ a × x + b × y = c $$
$$ d × x + e × y = f $$

```txt
Input: 1 3 -1 4 1 7
Output: 2 -1
```

{% endtab %}{% tab title='BJ_19532.py' %}

```py
a, b, c, d, e, f = map(int,input().split())
for x in range(-999,1000):
  for y in range(-999,1000):
    if a * x + b * y == c and d * x + e * y == f:
      print(x,y)
       break
```
