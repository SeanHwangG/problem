* Given X, Y, Z and x, y, z in each line
  * print E if y * 2 < Y
  * print D elif y < Y
  * print C elif x * 2 < X
  * print B elif x < X
  * print A otherwise

```txt
Input:
10 10 10
9 10 10

Output: B

Input:
10 10 10
0 4 10

Output: E
```

{% endtab %}{% tab title='BJ_20839.py' %}

```py
X, Y, Z = map(int, input().split())
x, y, z = map(int, input().split())

if y * 2 < Y: print('E')
elif y < Y: print('D')
elif x * 2 < X: print('C')
elif x < X: print('B')
else: print('A')
```
