* print two integers with lowest sum that satisfies given GCD, LCM

```txt
Input: 6 180
Output: 30 36
```

{% endtab %}{% tab title='BJ_2436.py' %}

```py
import math
g, l = map(int,input().split())
l //= g
for d in range(int(l ** .5), 0, -1):
  if l % d == 0 and math.gcd(d, l // d) == 1:
    print(g * d, g * (l // d))
    break
```
