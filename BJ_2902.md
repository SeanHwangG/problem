* Print only upper characters

```txt
Input: Knuth-Morris-Pratt
Output: KMP
```

{% endtab %}{% tab title='BJ_2902.go' %}

```go
package main

import "fmt"

func main() {
  var s string
  fmt.Scan(&s)
  for _, c := range s {
    if 'A' <= c && c <= 'Z' {
      fmt.Printf("%c", c)
    }
  }
  fmt.Println()
}
```
