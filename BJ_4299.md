* Given sum and difference, print two numbers with larger one first, Print -1 if they are impossible

```txt
Input: 3 1
Output: 2 1
```

{% endtab %}{% tab title='BJ_4299.py' %}

```py
a, b = map(int, input().split())
if a < b or (a + b) % 2 == 1:
  print(-1)
else:
  print((a + b) // 2, (a - b) // 2)
```
