* When the last digit is given a erased bit string
* the parity of this bit string is given, write a program to get the last digit correctly

```txt
Input:
101e
010010o
1e
000e
110100101o
#

Output:
1010
0100101
11
0000
1101001010
```

{% endtab %}{% tab title='BJ_4597.py' %}

```py
while 1:
  r = input()
  if r == '#':
    break
  print(r[:-1] + str(r.count('1') % 2 ^ (r[-1]=='o')))
```
