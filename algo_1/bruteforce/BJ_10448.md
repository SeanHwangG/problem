# [BJ_10448](https://acmicpc.net/problem/10448)

* en

  ```en
  Print if a number can be represented by the sum of following sequence
  1, 3, 6, 10, 15, 21 .

  ```

* tc

  ```tc
  Input: 3  # N
  10
  20
  1000

  Output: 1
  0
  1
  ```

## Solution

* py

  ```py
  triangle = [n*(n+1)//2 for n in range(1, 46)]
  eureka = [0] * 1001

  for i in triangle:
    for j in triangle:
      for k in triangle:
        if i+j+k <= 1000:
          eureka[i + j + k] = 1

  T = int(input())
  for _ in range(T):
    print(eureka[int(input())])
  ```
