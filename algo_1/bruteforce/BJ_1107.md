# [BJ_1107](https://acmicpc.net/problem/1107)



```txt
Input:
5457
3
6 7 8

Output: 6
```

## Solution

* py

  ```py
  n, m = int(input()), int(input())
  s = list(input().strip())
  result = abs(n - 100)
  for i in range(1000001):
    if all(n not in s for n in str(i)):
      result = min(result, len(str(i)) + abs(i - n))
  print(result)
  ```
