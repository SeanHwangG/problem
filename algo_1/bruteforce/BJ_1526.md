# [BJ_1526](https://acmicpc.net/problem/1526)

Print biggest number under N (4<=N<1000000), that consists of 4 or 7

```txt
Input: 100
Output: 77

Input: 75
Output: 74
```

## Solution

```py
n = int(input())
for i in range(n, 3, -1):
  if all([(s == '4' or s == '7') for s in str(i)]):
    print(i)
    break
```
