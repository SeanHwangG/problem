# [BJ_10844](https://acmicpc.net/problem/10844)

Count number of digits where different with adjacent is 1

```txt
Input: 1
Output: 9

Input: 2
Output: 17
```

## Solution

* py

  ```py
  a = b = c = d = e = 1
  for _ in range(int(input()) - 1):
    a, b, c, d, e = b, a + c, b + d, c + e, d + e
  print((a + 2 * (b + c + d + e)) % 10 ** 9)
  ```
