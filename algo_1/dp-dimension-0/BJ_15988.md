# [BJ_15988](https://acmicpc.net/problem/15988)

* en

  ```en
  When given an integer n (0 <= n =< 1,000,000), find number of methods to represent n as sum of 1, 2, and 3
  ```

* tc

  ```tc
  Input: 3
  4
  7
  10

  Output:
  7  # 1 + 3, 2 + 2, 3 + 1, 1 + 1 + 2, 1 + 2 + 1, 2 + 1 + 1, 1 + 1 + 1 + 1
  44
  274
  ```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  long long dp[1000001] = {1, 1, 2};
  int main()
  {
    int t, n;
    cin >> t;
    while (t--)
    {
      cin >> n;
      for (int i = 3; i <= n; i++)
        dp[i] = (dp[i - 1] + dp[i - 2] + dp[i - 3]) % (1000000009);
      cout << dp[n] << "\n";
    }
  }
  ```

* py
