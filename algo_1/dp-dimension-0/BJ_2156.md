# [BJ_2156](https://acmicpc.net/problem/2156)

You cannot step three consecutive coins
Print maximum coin you can get

```txt
Input: 6
6
10
13
9
8
1

Output: 33
```

## Solution

```py
a, b, c = 0, 0, 0
for i in range(int(input())):
  w = int(input())
  a, b, c = max(a, b, c), a + w, b + w
print(max(a,b,c))
```
