# [BJ_1463](https://acmicpc.net/problem/1463)

Divide by 3 or 2 (only divisible) or subtract 1
Find minimum operation

```txt
Input: 10
Output: 3
```

## Solution

* py

  ```py
  n = int(input()) + 1
  d = [0] * n
  for i in range(2, n):
    d[i] = min(d[i // 2] + i % 2, d[i // 3] + i % 3) + 1
  print(d[-1])
  ```
