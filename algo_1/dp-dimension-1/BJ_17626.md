# [BJ_17626](https://acmicpc.net/problem/17626)

Given N, find minimum number of squares to generate N

```txt
Input: 25
Output: 1
```

## Solution

* py

  ```py
  n = int(input())
  dp = [0] + [float('inf')] * n
  for n in range(n + 1):
    for root in range(int(n ** 0.5 + 1)):
      dp[n] = min(dp[n], dp[n - root ** 2] + 1)
  print(dp[-1])
  ```
