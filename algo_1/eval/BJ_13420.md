# [BJ_13420](https://acmicpc.net/problem/13420)

```en
If given formula is correct, output "correction" and "wrong answer" if incorrect
```

```txt
Input: 4
3 * 2 = 6
11 + 11 = 11
7 - 9 = -2
3 * 0 = 0

Output: correct
wrong answer
correct
correct
```

## Solution

* py

  ```py
  for _ in range(int(input())):
    eq, ans = input().split('=')
    print("correct" if eval(eq) == int(ans) else "wrong answer")
  ```
