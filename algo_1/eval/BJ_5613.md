# [BJ_5613](https://acmicpc.net/problem/5613)

* en

  ```en
  Make calcuator that can do +, -, / (throw away remainder), * operation
  ```

* tc

  ```tc
  Input: 1
  +
  1
  =

  Output: 2

  Input: 10
  -
  21
  *
  5
  =
  Output: 55

  Input: 100
  /
  3
  *
  3
  =
  Output: 99
  ```

## Solution

* cpp

  ```cpp
  #include <iostream>
  using namespace std;

  char op;
  int ans, x;
  int main() {
    cin >> ans >> op;
    while (1) {
      cin >> x;
      if (op == '+')ans += x;
      else if (op == '-') ans -= x;
      else if (op == '*') ans *= x;
      else if (op == '/') ans /= x;
      cin >> op;
      if (op == '=') break;
    }
    cout << ans;
  }
  ```

* py

  ```py
  c = input()
  while 1:
    line = input()
    if line == '=':
      print(c)
      break
    b = input()
    c = int(eval(str(c) + line + b))
  ```
