# [BJ_1547](https://acmicpc.net/problem/1547)

```en
Print location of cup at the end of simulation
```

```txt
Input: 4
3 1
2 3
3 1
3 2

Output: 3
```

## Solution

* py

  ```py
  n = int(input())
  cur = 1
  for _ in range(n):
    a, b = map(int, input().split())
    if cur == a:
      cur = b
    elif cur == b:
      cur = a
  print(cur)
  ```
