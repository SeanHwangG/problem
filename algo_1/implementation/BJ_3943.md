# [BJ_3943](https://acmicpc.net/problem/3943)

If n is even, divide by 2. If n is an odd number, multiply by 3 and add 1
Find biggest number from above sequence

```txt
Input: 4
1
3
9999
100000
Output:
1
16
101248
100000
```

## Solution

If n is even, divide by 2. If n is an odd number, multiply by 3 and add 1
Find biggest number from above sequence

```txt
4
1
3
9999
100000
1
16
101248
100000
```

```py
n_test = int(input())
for _ in range(n_test):
  n = int(input())

  mx = 1
  while n != 1:
    mx = max(mx, n)
    if n % 2 == 0:
      n //= 2
    else:
      n = n * 3 + 1
  print(mx)
```
