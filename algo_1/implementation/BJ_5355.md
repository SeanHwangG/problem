# [BJ_5355](https://acmicpc.net/problem/5355)

@ multiple 3, % add 5, # subtract 7

```txt
Input: 3
3 @ %
10.4 # % @
8 #

Output: 14.00
25.20
1.00
```

## Solution

```py
for t in range(int(input())):
  k = input().split()
  n = float(k[0])
  for i in range(1, len(k)):
    if k[i] == '@': n*=3
    if k[i] == '%': n+=5
    if k[i] == '#': n-=7
  print(f'{n:%.2f}')
```
