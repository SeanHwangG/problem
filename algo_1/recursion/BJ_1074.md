# [BJ_1074](https://acmicpc.net/problem/1074)

Print when row r, and column c will be visited

```txt
Input: 2 3 1
Output: 11
```

## Solution

* py

  ```py
  def recur(N, r, c):
    if r == c == 0:
      return 0
    side = 2 ** (N - 1)
    return recur(N - 1, r % side, c % side) + side ** 2 * (r // side * 2 + c // side)
  N, r, c = map(int, input().split())
  print(recur(N, r, c))
  ```
