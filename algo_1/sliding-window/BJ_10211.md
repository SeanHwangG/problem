# [BJ_10211](https://acmicpc.net/problem/10211)

Given X print maximum subarray

```txt
Input:
2
5
1 2 3 4 5
5
2 1 -2 3 -5

Output:
15
4
```

## Solution

* py

  ```py
  t = int(input())
  for _ in range(t):
    input()
    num = list(map(int, input().split()))
    tmp = res = num[0]
    for n in num[1:]:
      tmp = max(tmp + n, n)
      res = max(res, tmp)
    print(res)
  ```
