# [BJ_1783](https://acmicpc.net/problem/1783)

```en
If Night moves greater than 3, it has to take all possible moves
Given chessboard, find the maximum number of grids that knight can go
```

```txt
Input: 20 4
Output: 4
```

## Solution

* py

  ```py
  n, m = map(int, input().split())
  if n == 1:
    print(1)
  elif n == 2:
    print(min(4, (m + 1) // 2))
  elif m <= 6:
    print(min(4, m))
  else:
    print(m - 2)
  ```
