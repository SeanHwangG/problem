# [BJ_10212](https://acmicpc.net/problem/10212)

```en
Print Yonsei or Korea in equal probability
```

```txt
Output: Yonsei
```

## Solution

* cpp

  ```cpp
  #include <cstdio>
  #include <cstdlib>

  int main() {
    int a;
    srand((int)(unsigned long long)a);
    printf(rand() % 2? "Korea\n": "Yonsei\n");
    return 0;
  }
  }
  ```

* py

  ```py
  import random
  print(random.choice(['Yonsei','Korea']))
  ```
