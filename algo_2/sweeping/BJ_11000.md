# [BJ_11000](https://acmicpc.net/problem/11000)

Print min number of rooms to holds these meetings

```txt
Input:
3
1 3
2 4
3 5

Output: 2
```

## Solution

* py

  ```py
  import sys
  input = sys.stdin.readline

  n, lst = int(input()), []
  for _ in range(n):
    a,b = map(int, input().split())
    lst.append((a,1))
    lst.append((b,-1))
  lst.sort()
  ret , s =0, 0
  for a,b in lst:
    s += b
    ret = max(ret, s)
  print(ret)
  ```
