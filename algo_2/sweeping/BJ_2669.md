# [BJ_2669](https://acmicpc.net/problem/2669)

```en
Print total area covered by 4 rectangles
```

```txt
Input:
1 2 4 4
2 3 5 7
3 1 6 5
7 3 8 6

Output: 26
```

## Solution

* py

  ```py
  r = []
  for i in range(4):
    x, y, a, b = map(int, input().split())
    for m in range(x, a):
      [r.append((m, n)) for n in range(y,b)]
  print(len(set(r)))
  ```
