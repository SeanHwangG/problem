* Print sum of all 5 numbers

```txt
Input:
1
2
3
4
5

Output: 15
```

{% endtab %}{% tab title='BJ_5522.java' %}

```java
import java.util.Scanner;

class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int sum = 0;
    for (int i = 0; i < 5; i++)
      sum += sc.nextInt();
    System.out.println(sum);
  }
}
```
