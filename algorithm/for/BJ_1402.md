* Print yes for n times

```txt
Input: 3

Output:
yes
yes
yes
```

{% endtab %}{% tab title='BJ_1402.go' %}

```go
package main

import "fmt"
import "strings"

func main() {
  var t int
  fmt.Scan(&t)
  fmt.Println(strings.Repeat("yes\n", t))
}
```

{% endtab %}{% tab title='BJ_1402.py' %}

```py
for _ in range(int(input())):
  print('yes')
```
