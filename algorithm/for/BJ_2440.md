* Print Star in Following format

```txt
Input: 4

Output:
****
***
**
*
```

{% endtab %}{% tab title='BJ_2440.go' %}

```go
package main

import (
  "fmt"
  "strings"
)

func main() {
  n := 0
  fmt.Scanf("%d", &n)

  for n > 0 {
    fmt.Println(strings.Repeat("*", n))
    n -= 1
  }
}
```

{% endtab %}{% tab title='BJ_2440.py' %}

```py
a = int(input())
for i in range(1, a + 1):
  print("*" * (a - i + 1))
```
