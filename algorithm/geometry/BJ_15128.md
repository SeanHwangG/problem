* print if a \* c are divisible by b \* d * 2

```txt
Input:
15 1 28 3
Output: 1
```

{% endtab %}{% tab title='BJ_15128.py' %}

```py
a, b, c, d = map(int, input().split())
area = a * c / b / d / 2
if int(area) == area:
  print(1)
else:
  print(0)
```
