* Find the minimum sum of hamming distance from all points

```txt
Input:
4 4
1 2
1 4
3 1
4 2

Output: 8
```

{% endtab %}{% tab title='BJ_7571.py' %}

```py
n, m = map(int, input().split())

x = [0] * m
y = [0] * m
for i in range(m):
  x[i], y[i] = map(int, input().split())
x.sort()
y.sort()

px, py = x[m//2], y[m//2]

d = 0
for i in range(m):
  d += abs(px - x[i])
  d += abs(py - y[i])
print(d)
```
