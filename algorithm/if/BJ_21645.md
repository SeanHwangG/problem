* Given a, b, c, print a \* b if a < c else a \* ((c - 1) + b // c)

```txt
Input: 3 4 2
Output: 9
```

{% endtab %}{% tab title='BJ_21645.py' %}

```py
n, m, k = map(int, input().split())
print(n * m if m < k else n * ((k - 1) + m // k))
```
