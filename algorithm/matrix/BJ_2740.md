* Multiply two N \* M, M * K matrix

```txt
Input:
3 2
1 2
3 4
5 6
2 3
-1 -2 0
0 0 3

Output:
-1 -2 6
-3 -6 12
-5 -10 18
```

{% endtab %}{% tab title='BJ_2740.py' %}

```py
N, M = map(int, input().split())
A = [list(map(int, input().split())) for _ in range(N)]
M, K = map(int, input().split())
B = [list(map(int, input().split())) for _ in range(M)]
for n in range(N):
  for k in range(K):
    tmp = 0
    for m in range(M):
      tmp += A[n][m] * B[m][k]
    print(tmp, end = " ")
  print()
```
