* Print maximum number using only one multiplication and division

```txt
Input: 32 16 8
Output: 64
```

> Hint

* a // b x c doesn't work because remainder is lost

{% endtab %}{% tab title='BJ_15726.py' %}

```py
a, b, c = map(int, input().split())
print(max(a * b // c, a * c // b))
```
