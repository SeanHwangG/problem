* Convert to base-9 number system

```txt
Input: 100
Output: 121
```

{% endtab %}{% tab title='BJ_14491.py' %}

```py
N = int(input())
ret = count = 0
while N > 0:
  ret += N % 9 * pow(10, count)
  count += 1
  N //= 9
print(ret)
```
