* Given A, B, print A & B, A | B, A ^ B, ~A, ~B in each line

```txt
Input:
0001011000
0000101111

Output:
0000001000
0001111111
0001110111
1110100111
1111010000
```

{% endtab %}{% tab title='BJ_12813.py' %}

```py
a = input()
b = input()
a, b = int(a, 2), int(b, 2)
print(f'{a & b:0100000b}')
print(f'{a | b:0100000b}')
print(f'{a ^ b:0100000b}')
print(f'{~a & (1 << 100000) - 1:0100000b}')
print(f'{~b & (1 << 100000) - 1:0100000b}')
```
