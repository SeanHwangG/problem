* Print if sentence cna be abbreviated as UCPC

```txt
Input: Union of Computer Programming Contest club contest
Output: I love UCPC
```

{% endtab %}{% tab title='BJ_15904.py' %}

```py
import re
a = re.search('U.*C.*P.*C',input())
print('I love UCPC' if a else 'I hate UCPC')
```
