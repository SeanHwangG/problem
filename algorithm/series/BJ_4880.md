* Find if Arithmetic or geometric series

{% endtab %}{% tab title='BJ_4880.py' %}

```py
while True:
  a, b, c = map(int, input().split())
  if a == b == c == 0:
    break
  if c - b == b - a:
    print(f"AP {c + (b - a)}")
  else:
    print(f"GP {c * (b // a)}")
```
