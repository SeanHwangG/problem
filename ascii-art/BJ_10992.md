# [BJ_10992](https://acmicpc.net/problem/10992)

Print Star in Following format

```txt
Input: 4
Output:
   *
  * *
 *   *
*******
```

## Solution

```py
x = int(input())

for i in range(0, x):
  if i == 0:
    print(" " * (x - 1 - i) + "*")
  elif i < x - 1:
    print(" " * (x - 1 - i) + "*" + " " * ((i - 1) * 2 + 1) + '*')
  else:
    print("*" * (x * 2 - 1))
```
