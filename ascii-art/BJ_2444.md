# [BJ_2444](https://acmicpc.net/problem/2444)

Print star in following format

```txt
Input: 2
Output:
 *
***
 *
```

## Solution

```py
a = int(input())
for i in range(a - 1):
  print(' ' * (a - i - 1) + ('*' * (2 * i + 1)))
for i in range(a-1,-1,-1):
  print(' ' * (a - i - 1) + ('*' * (2 * i + 1)))
```
