# [BJ_1252](https://acmicpc.net/problem/1252)

Add number in binary

```txt
Input: 1001101 10010
Output: 1011111
```

## Solution

```js
console.log((require('fs').readFileSync('/dev/stdin')+'').trim().split(' ').map(x => {
  BigInt("0b" + x);
}).reduce((a, b) => a + b).toString(2))
```

```py
a, b = map(int, input().split(), [2, 2])
print(bin(a + b)[2:])
```
