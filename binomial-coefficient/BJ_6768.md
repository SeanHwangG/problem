# [BJ_6768](https://acmicpc.net/problem/6768)

Count number of ways to choose 3 people from N

```txt
Input: 4
Output: 1
```

## Solution

```cpp
#include <iostream>
using namespace std;

int main() {
  int n;
  cin >> n;
  cout << (n - 1) * (n - 2) * (n - 3) / 6;
}
```

```py
import math
n = int(input())
print(math.comb(n - 1, 3))
```
