# [BJ_2839](https://acmicpc.net/problem/2839)

Given 3, 5kg bags, find fewest number of bags to fill exactly N kg
Print -1 if you cannot fill exactly N kg

```txt
Input: 18    # N
Output: 4   # Three 5kg bags, one 3kg bag

Input: 4
Output: -1

Input: 9
Output: 3
```

## Solution

```py
N = int(input())
for i in range(0, N + 1, 3):
  if (N - i) % 5 == 0:
    print(i // 3 + (N - i) // 5)
    break
else:
  print(-1)
```
