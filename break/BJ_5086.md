# [BJ_5086](https://acmicpc.net/problem/5086)

In each line a, b are given and input ends with 0 0
Print whether a is factor or multiple of b, if not both, print neither

```txt
Input: 8 16
32 4
17 5
0 0
Output:
factor  # 16 is multiple of 8
multiple  # 32 is multiple of 4
neither
```

## Solution

```go
package main
import "fmt"

func main() {
  for {
    var a, b int
    fmt.Scan(&a, &b)
    if a == 0
      break
    if a % b == 0
      fmt.Println("multiple")
    else if b % a == 0
      fmt.Println("factor")
    else
      fmt.Println("neither")
  }
}
```

```py
while True:
  a, b = map(int, input().split())
  if a == 0 and b == 0:
    break
  if b % a == 0:
    print('factor')
  elif a % b == 0:
    print('multiple')
  else:
    print('neither')
```
