# [BJ_10610](https://acmicpc.net/problem/10610)

mix the numbers to make the largest number of times that are multiples of 30

```txt
Input: 102
Output: 210
```

## Solution

```py
li = list(map(int, input()))
if sum(li) % 3 != 0 or 0 not in li:
  print(-1)
else:
  print(*sorted(li, reverse=True), sep='')
```
