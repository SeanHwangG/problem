# [BJ_3035](https://acmicpc.net/problem/3035)

* en

  ```en
  Print magnified picture with r, c factor
  ```

* tc

  ```tc
  Input: 3 3 2 1  # row, col, mag_x, mag_y
  .x.
  x.x
  .x.

  Output:
  ..xx..
  xx..xx
  ..xx..
  ```

## Solution

* py

  ```py
  N, M, a, b = map(int, input().split())
  G = [input() for _ in range(N)]
  for i in range(N):
    for _ in range(a):
      for j in range(M):
        for _ in range(b):
          print(G[i][j], end='')
      print()
  ```
