# [BJ_10102](https://acmicpc.net/problem/10102)

```en
The winner of the final will be determined by a vote of the judges
Find out who wins when the judges' votes are given
```

```txt
Input: 6
ABBABB

Output: B
```

## Solution

* py

  ```py
  import collections
  input()
  co = collections.Counter(input())
  if co['A'] < co['B']:
    print('B')
  elif co['B'] < co['A']:
    print('A')
  else:
    print("Tie")
  ```
