# [BJ_1157](https://acmicpc.net/problem/1157)

* en

  ```en
  Given a word, print most commonly used alphabet (case insensitive) in capital letter
  If there are multiple alphabets that are used most frequently, output ?
  ```

* tc

  ```tc
  Input: Mississippi
  Output: ?

  Input: zZa
  Output: Z
  ```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  string s;
  int c[26];

  int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cin >> s;
    for (char i : s) c[toupper(i) - 'A']++;
    if (count(c, c + 26, *max_element(c, c + 26)) > 1)
      cout << "?\n";
    else
      cout << char(max_element(c, c + 26) - c + 'A') << endl;
  }
  ```

* java

  ```java
  import java.util.*;
    public static void main(String[] args) {
      Scanner sc = new Scanner(System.in);
      String L = sc.nextLine();
      L = L.toUpperCase();
      int[] arr = new int[26];
      for(int i=0; i < L.length(); i++)
        arr[L.charAt(i)-'A']++;

      int max = -1;
      char ch ='?';
      for(int i = 0; i < 26; i++) {
        if(arr[i] > max) {
          max = arr[i];
          ch = (char)(i+65);
        }
        else if (arr[i]==max)
          ch = '?';
      }
      System.out.println(ch);
    }
  }
  ```

* py

  ```py
  import collections
  st = input()
  cnter = collections.Counter(st.lower())
  mc = cnter.most_common(2)
  if len(mc) == 2 and mc[0][1] == mc[1][1]:
    print('?')
  else:
    print(mc[0][0].upper())
  ```
