# [BJ_11652](https://acmicpc.net/problem/11652)

* en

  ```en
  Prints the integer that you have the most
  ```

* tc

  ```tc
  Input:
  5
  1
  2
  1
  2
  1

  Output: 1
  ```

## Solution

* py

  ```py
  import sys
  from collections import Counter
  n_test = int(input())
  dic = Counter()
  top, ret = 0, 0
  for _ in range(n_test):
    n = int(sys.stdin.readline())
    dic[n]+=1
    if dic[n] > top or dic[n] == top and n < ret:
      top, ret = dic[n], n
  print(ret)
  ```
