# [BJ_10775](https://acmicpc.net/problem/10775)

Each person can sit on 1...n
Print maximum number of seats people can seat sequentially

```txt
Input: 4
3
4
1
1

Output: 2
```

## Solution

```py
def find(x):
  if x == p[x]:
    return x
  p[x] = find(p[x])
  return p[x]

G, P = int(input()), int(input())
p = [i for i in range(G+1)]
cnt = 0
for _ in range(P):
  g = int(input())
  scan_g = find(g)
  if scan_g == 0:
    break
  else:
    p[scan_g] = find(scan_g-1)
    cnt += 1
print(cnt)
```
