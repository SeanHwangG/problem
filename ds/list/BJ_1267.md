# [BJ_1267](https://acmicpc.net/problem/1267)

```en
Y: 10 won every 30s (starts from 1)
M: 15 won every 60s (starts from 1)
Print cheaper (both if they are same) one and their cost
```

```txt
Input: 30
40 40 40

Output: M 45  (Y =  20 + 20 + 20, M = 15 + 15 + 15)

Input: 3
61 61 61

Output: Y M 90

Input: 2
61 10
Output: Y 40  (Y = 30 + 10, M = 30 + 15)
```

## Solution

* go

  ```go
  package main

  import (
    "bufio"
    "fmt"
    "os"
  )

  var in = bufio.NewScanner(os.Stdin)

  func nextInt() int {
    in.Scan()
    r := 0
    for _, c := range in.Bytes() {
      r *= 10
      r += int(c - '0')
    }
    return r
  }
  func main() {
    in.Split(bufio.ScanWords)
    y, m := 0, 0
    for n := nextInt(); n > 0; n-- {
      v := nextInt()
      y += (v + 30) / 30 * 10
      m += (v + 60) / 60 * 15
    }
    switch {
    case y < m:
      fmt.Printf("Y %d", y)
    case m < y:
      fmt.Printf("M %d", m)
    default:
      fmt.Printf("Y M %d", y)
    }
  }
  ```

* js

  ```js
  const call = require("fs").readFileSync("/dev/stdin").toString().split("\n")[1].split(" ");

  let YC = 0, MC = 0;
  for (i of call) {
    YC += parseInt(i / 30) * 10 + 10;
    MC += parseInt(i / 60) * 15 + 15;
  }
  YC < MC
    ? console.log("Y", YC)
    : YC > MC
    ? console.log("M", MC)
    : (YC = MC ? console.log("Y", "M", YC) : 0);
  ```

* py

  ```py
  input()
  y = m = 0
  for p in map(int, input().split()):
    y += p // 30 * 10 + 10
    m += p // 60 * 15 + 15
  print('M' if y > m else 'Y' if y<m else 'Y M', min(y, m))
  ```
