# [BJ_5235](https://acmicpc.net/problem/5235)

```en
For N lines, print whether sum of even is greater than sum of odd numbers
```

```kr
한 줄에 짝수의 개수가 많으면 EVEN, 홀수가 많으면 ODD, 같으면 TIE를 출력하라
```

```txt
Input: 4  # N
8 5 7 2 1 10 13 6 12  # First number is number of element in the list
4 5 2 3 4
8 1 2 1 2 1 1 1 1
6 1 1 2 3 5 8

Output: EVEN
ODD
ODD
TIE  # 2 + 8 = 1 + 1 + 3 + 5
```

## Solution

* go

  ```go
  package main

  import (
    "bufio"
    "os"
  )

  var in = bufio.NewScanner(os.Stdin)

  func nextInt() int {
    in.Scan()
    r := 0
    for _, c := range in.Bytes() {
      r *= 10
      r += int(c - '0')
    }
    return r
  }

  func main() {
    in.Split(bufio.ScanWords)
    for t := nextInt(); t > 0; t-- {
      os.Stdout.WriteString(main2())
    }
  }

  func main2() string {
    r1, r2 := 0, 0
    for n := nextInt(); n > 0; n-- {
      if v := nextInt(); v%2 == 0 {
        r2 += v
      } else {
        r1 += v
      }
    }
    switch {
    case r1 > r2:
      return "ODD\n"
    case r1 < r2:
      return "EVEN\n"
    }
    return "TIE\n"
  }
  ```

* py

  ```py
  t = int(input())
  for i in range(t):
    o = e = 0
    for j in list(map(int, input().split()))[1:]:
      if j % 2: o += j
      else: e += j
    if o > e: print('ODD')
    elif o < e: print('EVEN')
    else: print('TIE')
  ```
