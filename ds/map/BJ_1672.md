# [BJ_1672](https://acmicpc.net/problem/1672)

```en
Given rules print result sequence

| Before | After |
| ------ | ----- |
| 'AA'   | 'A',  |
| 'AG'   | 'C',  |
| 'AC'   | 'A',  |
| 'AT'   | 'G',  |
| 'GA'   | 'C',  |
| 'GG'   | 'G',  |
| 'GC'   | 'T',  |
| 'GT'   | 'A',  |
| 'CA'   | 'A',  |
| 'CG'   | 'T',  |
| 'CC'   | 'C',  |
| 'CT'   | 'G',  |
| 'TA'   | 'G',  |
| 'TG'   | 'A',  |
| 'TC'   | 'G',  |
| 'TT'   | 'T',  |
```

```txt
Input: 6
AAGTCG

Output: A  # AGTCG -> CTCG -> GCG -> TG -> A
```

## Solution

* go

  ```go
  package main

  import (
    "bufio"
    "os"
  )

  var in = bufio.NewScanner(os.Stdin)
  var data = map[byte]map[byte]byte{
    'A': {'A': 'A', 'G': 'C', 'C': 'A', 'T': 'G'},
    'G': {'A': 'C', 'G': 'G', 'C': 'T', 'T': 'A'},
    'C': {'A': 'A', 'G': 'T', 'C': 'C', 'T': 'G'},
    'T': {'A': 'G', 'G': 'A', 'C': 'G', 'T': 'T'},
  }

  func next() []byte {
    in.Scan()
    return in.Bytes()
  }

  func main() {
    in.Buffer(make([]byte, 1048576), 1048576)
    _, b := next(), next()
    for i := len(b) - 2; i >= 0; i-- {
      b[i] = data[b[i]][b[i+1]]
    }
    os.Stdout.Write(b[:1])
  }
  ```
