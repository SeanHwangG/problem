# [BJ_10867](https://acmicpc.net/problem/10867)

Print in unique element in ascending order

```txt
Input:
10
1 4 2 3 1 4 2 3 1 2

Output: 1 2 3 4
```

## Solution

* cpp

  ```cpp
  #include <iostream>
  #include <set>

  using namespace std;

  int main() {
    int n, a;
    cin >> n;
    set<int> S;
    while (n--) {
      cin >> a;
      S.insert(a);
    }
    for (auto c : S) cout << c << " ";
  }
  ```

* py

  ```py
  input()
  print(*sorted(set(map(int, input().split()))))
  ```
