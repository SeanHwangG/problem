  If num == 0 print value with min absolute value

## Solution

* py

  ```py
  from heapq import *
  n, *m = map(int,[*open(0)])
  l = []
  for i in m:
    heappush(l,(abs(i),i)) if i else print(len(l) and heappop(l)[1])
  ```
