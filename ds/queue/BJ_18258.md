# [BJ_18258](https://acmicpc.net/problem/18258)

* en

  ```en
  Design queue
  front: returns first element
  back: returns last element
  empty: returns 1 if size == 0 else 0
  size: returns size of queue
  pop: returns first element, returns -1 if there is no element
  ```

* tc

  ```tc
  Input: 15
  push 1
  push 2
  front
  back
  size
  empty
  ```

## Solution

* cc

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;
  queue <int> q;
  int t, x;
  string cmd;
  int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cin >> t;
    while (t--) {
      cin >> cmd;
      if (cmd == "push") {
        cin >> x;
        q.push(x);
        continue;
      }
      else if (cmd == "size") x = q.size();
      else if (cmd == "empty") x = q.empty();
      else if (q.empty()) x = -1;
      else if (cmd == "back") x = q.back();
      else {
        x = q.front();
        if (cmd == "pop") q.pop();
      }
      cout << x << '\n';
    }
  }
  ```

* py

  ```py
  from collections import deque
  import sys
  N = int(input())
  samples = deque()
  for _ in range(N):
    tokens = sys.stdin.readline().split()
    if tokens[0] == 'push':
      samples.append(tokens[1])
    elif tokens[0] == 'pop':
      print(samples.popleft() if samples else -1)
    elif tokens[0] ==  'size':
      print(len(samples))
    elif tokens[0] == 'empty':
      print(0 if samples else 1)
    elif tokens[0] == 'front':
      print(samples[0] if samples else -1)
    elif tokens[0] == 'back':
      print(samples[-1] if samples else -1)
  ```
