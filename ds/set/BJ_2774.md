# [BJ_2774](https://acmicpc.net/problem/2774)

* en

  ```en
  Find number of unique numbers
  ```

* tc

  ```tc
  Input:
  3
  1
  1
  2

  Output: 2
  ```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  int main() {
    int T; cin >> T;
    while (T--) {
      unordered_set<char> result;
      string x; cin >> x;
      for (int i = 0; i < x.size(); i++)
        result.insert(x[i]);
      cout << result.size() << '\n';
    }
    return 0;
  }
  ```

* py

  ```py
  for _ in range(int(input())):
    print(len(set(input())))
  ```
