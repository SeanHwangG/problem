# [BJ_9935](https://acmicpc.net/problem/9935)

* en

  ```en
  Keep remove bomb string, if none left print FRULA
  ```

* tc

  ```tc
  Input:
  mirkovC4nizCC44
  C4

  Output: mirkovniz

  Input:
  12ab112ab2ab
  12ab

  Ouput: FRULA
  ```

## Solution

* py

  ```py
  word, bomb = input(), list(input())
  res = []
  for char in word:
    res.append(char)
    if res[-len(bomb):] == bomb:
      res[-len(bomb):] = []
  print(''.join(res) if res else "FRULA")
  ```
