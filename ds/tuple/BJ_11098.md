# [BJ_11098](https://acmicpc.net/problem/11098)

```en
For N testcase, playersgiven list of salary, name, print the most expensive player
```

```txt
Input: 2  # N
3  # Number of players
10 Iversen
1000000 Nannskog
2000000 Ronaldinho
2. # Number of players
1000000 Maradona
999999 Batistuta

Output: Ronaldinho
Maradona
```

## Solution

* cpp

  ```cpp
  #include <iostream>
  #include <vector>
  #include <string>
  using namespace std;
  int main() {
    int n, p;
    cin >> n;
    while (n--) {
      cin >> p;
      vector <pair<int, string>> s(p);
      int ans = 0, i = 0, ex = 0;
      while (p--) {
        cin >> s[i].first >> s[i].second;
        if (s[i].first > ex) {
          ex = s[i].first;
          ans = i;
        }
        i++;
      }
      cout << s[ans].second << '\n';
    }
  }
  ```
