# [BJ_3036](https://acmicpc.net/problem/3036)

Given n and a, b1, b2..., bn
Print b1 / a, b2 / a in irreducible fraction

```txt
Input:
4
12 3 8 4

Output:
4/1
3/2
3/1
```

## Solution

```py
import math
input()
f, *g = map(int, input().split())
for i in g:
  c = math.gcd(i,f)
  print(f'{f // c}/{i // c}')
```
