# [BJ_10994](https://acmicpc.net/problem/10994)



```txt
Input: 3
Output:
*********
*       *
* ***** *
* *   * *
* * * * *
* *   * *
* ***** *
*       *
*********
```

## Solution

```py
n = int(input().strip())
x = 4 * n - 3

for i in range(x):
  l = ''
  for j in range(x):
    if max(abs(i - 2 * n + 2), abs(j - 2 * n + 2)) % 2 == 0:
      l += '*'
    else:
      l += ' '
  print(l)
```
