# [BJ_11161](https://acmicpc.net/problem/11161)



```txt
Input: 1
3
3 5
3 4
1 0
Output: 3
```

## Solution

```py
for _ in range(int(input())):
  max_people, cur_people = 0, 0
  for _ in range(int(input())):
    a, b = map(int, input().split())
    cur_people = cur_people + b - a
    max_people = max(cur_people, max_people)
  print(max_people)
```
