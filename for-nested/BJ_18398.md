# [BJ_18398](https://acmicpc.net/problem/18398)

Print sum and multiple for N test for n line

```txt
Input: 1
2
20 30
40 60

Output:
50 600
100 2400
```

## Solution

```py
for _ in range(int(input())):
  for _ in range(int(input())):
    a, b = map(int, input().split())
    print(a + b, a * b)
```
