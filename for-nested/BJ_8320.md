# [BJ_8320](https://acmicpc.net/problem/8320)

Sanggeun has n squares with a side length of 1
How many rectangles can be made using this square in total?

```txt
Input: 6
Output: 8
```

## Solution

```py
n = int(input())
cnt = 0
for i in range(1, n + 1):
  for j in range(i, n + 1):
    if i * j <= n:
      cnt += 1
print(cnt)
```
