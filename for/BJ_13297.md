# [BJ_13297](https://acmicpc.net/problem/13297)

Print length of string for each N line

```txt
Input: 2  # N
ab
a
Output: 2
1
```

## Solution

```cpp
#include <iostream>
#include <string>

using namespace std;
int main() {
  int T;
  string in;
  for (scanf("%d", &T); T--;) {
    cin >> in;
    cout << in.size() << endl;
  }
  return 0;
}
```

```py
for _ in range(int(input())):
  print(len(input()))
```
