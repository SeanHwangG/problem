# [BJ_2720](https://acmicpc.net/problem/2720)

For N lines, print number of exchanges in Quarter, Dime, Nickel, Penny

```txt
Input: 3  # N
124
25
194
Output: 4 2 0 4  # 25 * 4 + 10 * 2 + 5 * 0 + 1 * 4
1 0 0 0
7 1 1 4
```

## Solution

```py
for i in range(int(input())):
  x = int(input())
  print(x // 25, x % 25 // 10, x % 25 % 10 // 5, x % 25 % 10 % 5)
```
