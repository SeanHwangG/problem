# [BJ_2563](https://acmicpc.net/problem/2563)

Print total area of overlapping region

```txt
Input:
3
3 7
15 7
5 2

Output: 260
```

## Solution

* py

  ```py
  n = int(input())
  x = set()
  for k in range(n):
    a, b = map(int,input().split())
    for i in range(10):
      for j in range(10):
        x.add((a+i, b+j))
  print(len(x))
  ```
