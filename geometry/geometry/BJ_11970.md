# [BJ_11970](https://acmicpc.net/problem/11970)

```en
Print total length of two lines

```

```kr
두 선분이 있을때 두 선분이 차지하는 총 길이를 구하라
```

```txt
Input:
7 10
4 8

Output: 6
```

## Solution

* py

  ```py
  a, b = map(int, input().split())
  c, d = map(int, input().split())
  if a > c:
    a, b, c, d = c, d, a, b
  print(b - a + d - c - max(0, min(d, b) - c))
  ```
