# [BJ_15128](https://acmicpc.net/problem/15128)

```en
Print if a x c are divisible by b x d x 2
```

```kr
a, b, c, d가 주어진다
a * c 가 b * d * 2 로 나누어 떨어지면 1
그렇지 않으면 0을 출력하라.
```

```txt
Input:
15 1 28 3
Output: 1
```

## Solution

* py

  ```py
  a, b, c, d = map(int, input().split())
  area = a * c / b / d / 2
  if int(area) == area:
    print(1)
  else:
    print(0)
  ```
