# [BJ_20215](https://acmicpc.net/problem/20215)

```en
Given rectangle print width + height - diagonal
```

```kr
사각형의 가로와 세로가 주어진다
가로와 세로의 합에서 대각선의 길이를 뺀 값을 출력하라
```

```txt
Input: 3 4
Output: 2
```

## Solution

* py

  ```py
  a, b = map(int, input().split())
  print(a + b - (a * a + b * b) ** .5)
  ```
