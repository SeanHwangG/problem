# [BJ_5073](https://acmicpc.net/problem/5073)

```en
Print (Equilateral, Isosceles, Scalene, Invalid) for each input
```

```txt
Input: 7 7 7
6 5 4
3 2 5
6 2 6
0 0 0
Output:
Equilateral
Scalene
Invalid
Isosceles
```

## Solution

* py

  ```py
  while True:
    A, B, C = map(int, input().split())
    if A == B == C == 0:
      break
    if max(A, B, C) >= A + B + C - max(A, B, C):
      print("Invalid")
    elif A == B == C:
      print("Equilateral")
    elif A == B or B == C or A == C:
      print("Isosceles")
    else:
      print("Scalene")
  ```
