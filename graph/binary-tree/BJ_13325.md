# [BJ_13325](https://acmicpc.net/problem/13325)

```en
Given binary tree, you want to make sum path to root same
Print sum of all nodes in result tree
```

```txt
Input:
2
2 2 2 1 1 3

Output: 15  # 3 2 2 2 3 3
```

## Solution

* cpp

  ```cpp
  #include<bits/stdc++.h>
  using namespace std;
  int k, a[1 << 21], r, i, t;
  int main() {
    cin >> k;
    for (i = 2; i < 1 << k + 1; i++) scanf("%d", a + i);
    while (i -= 2) {
      r += t = max(a[i], a[i + 1]);
      a[i / 2] += t;
    }
    cout << r + t;
    return 0;
  }
  ```
