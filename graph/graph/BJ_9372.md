Prints min number of flights that Sang-geun must take to travel to all countries

```txt
Input:
2
3 3
1 2
2 3
1 3
5 4
2 1
2 3
4 3
4 5

2
4
```

* py

  import sys
  input = sys.stdin.readline
  T = int(input())
  for _ in range(T):
    N, M = map(int, input().split())
    for _ in range(M):
      a, b = map(int, input().split())
