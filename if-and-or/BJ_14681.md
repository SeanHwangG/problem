# [BJ_14681](https://acmicpc.net/problem/14681)

Given x, y print corresponding quadrant

```txt
Input: 12
5
Output: 1

Input: 9
-13
Output: 4
```

## Solution

```py
a = int(input())
b = int(input())
if a > 0 and b > 0:
  print(1)
elif a < 0 and b > 0:
  print(2)
elif a < 0 and b < 0:
  print(3)
else:
  print(4)
```

```sh
read x
read y

if [ $x -gt 0 ] && [ $y -gt 0 ]; then
  echo 1
elif [ $x -lt 0 ] && [ $y -gt 0 ]; then
  echo 2
elif [ $x -lt 0 ] && [ $y -lt 0 ]; then
  echo 3
else
  echo 4
fi
```
