```py
A, B = map(int, input().split())
print(">" if A > B else "<" if A < B else "==")
```

```r
x=scan("stdin")
a=x[1]
b=x[2]
cat(ifelse(a>b, ">", ifelse(a < b, "<", ifelse(a==b, "=="))))
```

```sh
read a b
if [ $a -eq $b ]; then
  echo '=='
elif [ $a -gt $b ]; then
  echo '>'
else
  echo '<'
fi
```
