# [BJ_20839](https://acmicpc.net/problem/20839)

Given X, Y, Z and x, y, z in each line
print E if y * 2 < Y
print D elif y < Y
print C elif x * 2 < X
print B elif x < X
print A otherwise

```txt
Input: 10 10 10  # X, Y, Z
9 10 10          # x, y, z
Output: B

Input: 10 10 10
0 4 10
Output: E
```

## Solution

```py
X, Y, Z = map(int, input().split())
x, y, z = map(int, input().split())

if y * 2 < Y: print('E')
elif y < Y: print('D')
elif x * 2 < X: print('C')
elif x < X: print('B')
else: print('A')
```
