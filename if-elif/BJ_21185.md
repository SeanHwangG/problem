# [BJ_21185](https://acmicpc.net/problem/21185)

Pick n numbers between 1 to 100
Print oddity of the sum of consecutive n numbers, (Either if it can be both even and odd)

```txt
Input: 2    # n
Output: Odd

Input: 1
Output: Either

Input: 4
Output: Even
```

## Solution

```py
n = int(input())
if n % 4 == 0:
  print('Even')
elif n % 2 == 0:
  print('Odd')
else:
  print('Either')
```
