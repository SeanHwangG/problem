# [BJ_16953](https://acmicpc.net/problem/16953)

Print minimum number of operations to convert a to b
Add one at the end
Multiple 2

```txt
Input: 100 40021
Output: 5
```

## Solution

```py
a, b = map(int, input().split())
c = 0
while b > a:
  c += 1
  if b % 10 == 1:
    b //= 10
  elif b % 2:
    break
  else:
    b //= 2
print(c + 1 if b == a else -1)
```
