# [BJ_17256](https://acmicpc.net/problem/17256)

A x B = (a.z + b.x, a.y Ã— b.y, a.x + b.z)
Given A, C where A x B = C, find B

```txt
Input: 15 16 17
19 32 90
Output: 2 2 75
```

## Solution

```py
ax, ay, az = map(int, input().split())
cx, cy, cz = map(int, input().split())
print(cx - az, cy // ay, cz - ax)
```
