# [BJ_10871](https://acmicpc.net/problem/10871)

Prints all numbers smaller than X in A

```txt
Input: 10 5
1 10 4 9 2 3 8 5 7 6

Output: 1 4 2 3
```

## Solution

```cpp
#include <iostream>

using namespace std;

int main() {
  int n, x;
  cin >> n >> x;
  while (n--) {
    int a;
    cin >> a;
    if (a < x)
      cout << a << " ";
  }
}
```

```go
package main

import "fmt"

func main() {
  var a, b, c int
  fmt.Scan(&a, &b)

  for i := 0; i < a; i++ {
    fmt.Scan(&c)
    if c < b {
      fmt.Printf("%d ", c)
    }
  }
}
```
