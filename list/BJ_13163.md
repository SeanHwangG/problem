# [BJ_13163](https://acmicpc.net/problem/13163)

For N numbers, delete until first blank and print concatnated string with God

```txt
Input: 5  # N
baek joon
koo sa ga
ac ka
yu ka ri ko
ke sa ki yo

Output: godjoon
godsaga
godka
godkariko
godsakiyo
```

## Solution

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
string s;
int main() {
  cin >> n;
  while(n--) {
    getline(cin, s);
    s.replace(s.begin(), s.begin() + s.find_first_of(' '), "god");
    for(char c : s) if (c != ' ') cout << c;
    cout << '\n';
  }
}
```
