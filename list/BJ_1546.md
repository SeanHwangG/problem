# [BJ_1546](https://acmicpc.net/problem/1546)

Print average score devided by max score multiplied by length of list

```txt
Input: 3
40 80 60

Output: 75.0  # (40 + 80 + 60) / 80 * 3
```

## Solution

```py
n = int(input())
li = list(map(int, input().split()))
print(sum(li) / max(li) * 100 / len(li))
```
