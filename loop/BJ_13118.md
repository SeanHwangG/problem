# [BJ_13118](https://acmicpc.net/problem/13118)

Print # person who hits with falling apple
If no one conflicts with an apple, print 0 instead

```txt
Input:
-5 -2 1 5
1 4 2

Output : 3
```

## Solution

```py
L = list(map(int, input().split()))
x, _, _ = map(int, input().split())
if x in L:
  print(L.index(x) + 1)
else:
  print(0)
```
