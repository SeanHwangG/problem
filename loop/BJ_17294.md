# [BJ_17294](https://acmicpc.net/problem/17294)

Determine if the number is in arithmetic sequence

```txt
Input: 631
Output: 흥칫뿡!! <(￣ ﹌ ￣)>
```

## Solution

```py
n = input()

if all([int(n[k]) - int(n[k - 1]) == int(n[1]) - int(n[0]) for k in range(1, len(n))]):
  print("◝(⑅•ᴗ•⑅)◜..°♡ 뀌요미!!")
else:
  print("흥칫뿡!! <(￣ ﹌ ￣)>")
```
