# [BJ_2754](https://acmicpc.net/problem/2754)

Given letter score, print GPA

```txt
Input: A0
Output: 4.0
```

## Solution

```cpp
#include <iostream>
using namespace std;

int main() {
  char a, b; cin >> a >> b;
  cout.precision(1);
  if (a != 'F')
    cout << fixed << (4 - (a - 'A') + (b == '+' ? 0.3 : (b == '0' ? 0 : -0.3)));
  else
    cout << "0.0";
}
```

```go
package main

import "fmt"

func main() {
  var s string
  fmt.Scan(&s)
  switch s {
  case "A+":
    fmt.Println("4.3")
  case "A0":
    fmt.Println("4.0")
  case "A-":
    fmt.Println("3.7")
  case "B+":
    fmt.Println("3.3")
  case "B0":
    fmt.Println("3.0")
  case "B-":
    fmt.Println("2.7")
  case "C+":
    fmt.Println("2.3")
  case "C0":
    fmt.Println("2.0")
  case "C-":
    fmt.Println("1.7")
  case "D+":
    fmt.Println("1.3")
  case "D0":
    fmt.Println("1.0")
  case "D-":
    fmt.Println("0.7")
  default:
    fmt.Println("0.0")
  }
}
```
