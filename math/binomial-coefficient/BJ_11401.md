# [BJ_11401](https://acmicpc.net/problem/11401)

Given N, K print N choose K mod 1,000,000,007

```txt
Input: 5 2
Output: 10
```

## Solution

* py

  ```py
  n, k = map(int, input().split())
  MOD = 1000000007
  res = 1
  k = min(k, n - k)
  for i in range(k):
    res = res * (n - i) % MOD
    res = res * pow(i + 1, MOD - 2, MOD) % MOD
  print(res)
  ```
