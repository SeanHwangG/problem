# [BJ_6591](https://acmicpc.net/problem/6591)

```en
How many ways do you choose k out of n elements without order?
```

```txt
Input:
4 2
10 5
49 6
0 0

Output:
6
252
13983816
```

## Solution

* py

  ```py
  import math
  while True:
    a, b = map(int, input().split())
    if a == b == 0:
      break
    print(math.comb(a, b))
  ```
