# [BJ_3460](https://acmicpc.net/problem/3460)

```en
Find all 1's position in binary
```

```txt
Input: 1
13

Output: 0 2 3
```

## Solution

* js

  ```js
  n = require('fs').readFileSync('dev/stdin').toString().trim().split('\n')
  n.slice(1).map(v => +v).forEach(v => {
    let b = v.toString(2)
    console.log(b.split('').reverse().reduce((t, v, i)=>{
      if (v == 1) return [...t, i]
      return t
    },[]).join(' '))
  })
  ```

* py

  ```py
  n = int(input())
  for _ in range(n):
    x = int(input())
    n = 0
    while x != 0:
      if x & 1:
        print(n, end=' ')
      n += 1
      x //= 2
    print()
  ```
