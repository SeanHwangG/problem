# [BJ_11050](https://acmicpc.net/problem/11050)

* en

  ```en
  Find result of binomial coefficient (number of ways to choose k numbers from n without order)
  ```

* tc

  ```tc
  Input: 5 2
  Output: 10
  ```

## Solution

* cpp

  ```cpp
  #include <cstdio>
  int main() {
    int n, k, a = 1, b = 1;
    scanf("%d%d", &n, &k);
    while (k) {
      a *= n--;
      b *= k--;
    }
    printf("%d", a / b);
  }
  ```

* py

  ```py
  import math
  a, b = map(int, input().split())
  print(math.comb(a, b))
  ```
