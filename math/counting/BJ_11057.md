# [BJ_11057](https://acmicpc.net/problem/11057)

```en
Find N length ascending number
```

```txt
Input: 3
Output: 220
```

## Solution

* cpp

  ```cpp
  #include <stdio.h>
  int D[1020][10];
  int main(){
    int i,j,n;
    scanf("%d",&n);
    for(i=0;i<n+10;++i){
      D[i][0]=1;
      for(j=1;j<10&&j<=i;++j)D[i][j]=(D[i-1][j]+D[i-1][j-1])%10007;
    }
    printf("%d",D[n+9][9]);
    return 0;
  }
  ```

* py

  ```py
  import math
  n = int(input())
  print(math.comb(9 + n, n) % 10007)
  ```
