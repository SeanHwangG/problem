# [BJ_15439](https://acmicpc.net/problem/15439)

You have N shirts and pants with distict color
Find the number of ways to wear without wearing same color of shirt and part

```txt
Input: 3
Output: 6
```

## Solution

```sh
read a
echo $((a * a - a))
```

```py
a = int(input())
print(a * (a - 1))
```
