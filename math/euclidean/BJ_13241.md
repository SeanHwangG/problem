# [BJ_13241](https://acmicpc.net/problem/13241)

Find LCM of each pairs

```txt
Input: 3 5
Output: 15
```

## Solution

* py

  ```py
  def GCD(a, b):
    while(b):
      a, b = b, a % b
    return a
  def LCM(a, b):
    return a * b // GCD(a, b)
  a, b = map(int, input().split())
  print(LCM(a, b))
  ```
