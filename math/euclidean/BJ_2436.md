# [BJ_2436](https://acmicpc.net/problem/2436)

Print 2 ints with lowest sum that satisfies given GCD, LCM

```txt
Input: 6 180
Output: 30 36
```

## Solution

```py
import math
g, l = map(int,input().split())
l //= g
for d in range(int(l ** .5), 0, -1):
  if l % d == 0 and math.gcd(d, l // d) == 1:
    print(g * d, g * (l // d))
    break
```
