# [BJ_2609](https://acmicpc.net/problem/2609)

Find GCD and LCM of each pair

```txt
Input: 24 18
Output: 6
72
```

## Solution

```py
def GCD(a, b):
   while(b):
     a, b = b, a % b
   return a
def LCM(a, b):
  return a * b // GCD(a, b)
a, b = map(int, input().split())
print(GCD(a, b), LCM(a, b))
```
