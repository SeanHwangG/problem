# [BJ_10826](https://acmicpc.net/problem/10826)

Print Nth fibonacci number

```txt
Input: 10
Output: 55
```

## Solution

* js

```js
let num = +require('fs').readFileSync('/dev/stdin').toString();
let dp = [0, 1];
for (let i = 2; i <= num; i++) {
  dp[i] = BigInt(dp[i-1]) + BigInt(dp[i-2]);
}
console.log(dp[num].toString());
```

* py

```py
n = int(input())
a, b = 1, 0
for i in range(n):
  a, b = b, a + b
print(b)
```
