# [BJ_15624](https://acmicpc.net/problem/15624)

Find nth fibonacci number (N < 1,000,000)

```txt
Input: 0
Output: 0
```

## Solution

* py

```py
def mul(a,b):
  return [[sum(i * j for i, j in zip(x, y)) % 1000000007 for y in b] for x in a]
def power(a,n):
  if n > 1:
    x = power(a, n//2)
    x = mul(x,x)
    if n % 2 != 0:
      return mul(x,a)
    else:
      return(x)
  else:
    return(a)

print(power([[1,1],[1,0]],int(input()))[1][0])
```
