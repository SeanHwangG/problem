# [BJ_2749](https://acmicpc.net/problem/2749)

```en
Find nth fibonacci number
```

```txt
Input: 1000
Output: 228875
```

## Solution

* py

  ```py
  n = int(input())
  def fibo3(n):
    a, b = 0, 1
    for _ in range(n):
      a, b = b%1000000, (a+b)%1000000
    return a
  print(fibo3(n % (15 * (10 ** 5))))
  ```
