# [BJ_13900](https://acmicpc.net/problem/13900)

Find sum of multiple in every pair

```txt
Input: 4
1 2 3 4

Output: 35

Input: 3
2 3 4
Output: 26
```

## Solution

```py
N = int(input())
nums = list(map(int, input().split()))
sum_ = sum(nums)
result = 0
for num in nums:
  sum_ -= num
  result += num * sum_
print(result)
```
