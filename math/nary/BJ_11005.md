# [BJ_11005](https://acmicpc.net/problem/11005)

Print decimal into B's number

```txt
Input: 60466175 36
Output: ZZZZZ
```

## Solution

* py

  ```py
  a, b = map(int, input().split())
  conv = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
  ret = ""
  while a != 0:
    ret = conv[a % b] + ret
    a //= b
  print(ret)
  ```
