# [BJ_1373](https://acmicpc.net/problem/1373)

Convert binary into oct

```txt
Input: 11001100
Output: 314
```

## Solution

* go

  ```go
  package main

  import (
    "bufio"
    "fmt"
    "math/big"
    "os"
  )

  var in = bufio.NewReader(os.Stdin)

  func main() {
    r := new(big.Int)
    fmt.Fscanf(in, "%b", r)
    fmt.Printf("%o\n", r)
  }
  ```

* py

  ```py
  print(oct(int(input(), 2))[2:])
  ```
