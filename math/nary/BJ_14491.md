# [BJ_14491](https://acmicpc.net/problem/14491)

Convert to base-9 number system

```txt
Input: 100
Output: 121
```

## Solution

```go
package main

import (
  "fmt"
  "strconv"
)

func main() {
  var t int64
  fmt.Scan(&t)
  fmt.Println(strconv.FormatInt(t, 9))
}
```

```py
N = int(input())
ret = count = 0
while N > 0:
  ret += N % 9 * pow(10, count)
  count += 1
  N //= 9
print(ret)
```
