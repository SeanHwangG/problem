# [BJ_15989](https://acmicpc.net/problem/15989)

Given integer n, find # ways to represent n as the sum of 1, 2, and 3

```txt
Input:
3
4
7
10

Output:
4
8
14
```

## Solution

* py

```py
dp = [1 for i in range(10001)]
for i in range(2, 10001):
  dp[i] += dp[i - 2]

for i in range(3, 10001):
  dp[i] += dp[i - 3]

for _ in range(int(input())):
  print(dp[int(input())])
```
