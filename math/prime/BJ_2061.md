# [BJ_2061](https://acmicpc.net/problem/2061)

```en
Print GOOD if prime else BAD
```

```txt
Input: 143 11
Output: GOOD  # 2, 3 ... 10 doesn't divide 143
```

## Solution

* py

  ```py
  p, k = map(int,input().split())
  for i in range(2, k):
    if p % i == 0:
      print("BAD", i)
      break
  else:
    print("GOOD")
  ```
