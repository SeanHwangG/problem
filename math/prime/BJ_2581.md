# [BJ_2581](https://acmicpc.net/problem/2581)

Given numbers M, N, find sum and min of these few numbers by selecting all natural numbers above M and below N

```txt
Input: 60
100

Output: 620
61
```

## Solution

* py

  ```py
  M, N = int(input()), int(input())
  li = []
  for n in range(M, N + 1):
    if is_prime(n):
      li.append(n)

  if len(li) == 0:
    print(-1)
  else:
    print(sum(li))
    print(min(li))
  ```
