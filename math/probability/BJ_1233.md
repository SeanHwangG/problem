# [BJ_1233](https://acmicpc.net/problem/1233)

Given three dice print minimum value that has highest probability

```txt
Input: 3 2 3
Output: 5
```

## Solution

* py

  ```py
  a, b, c = map(int, input().split())
  l = [0] * 81
  for i in range(a):
    for j in range(b):
      for k in range(c):
        l[i+j+k]+=1
  print(l.index(max(l))+3)
  ```
