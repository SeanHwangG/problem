# [BJ_11522](https://acmicpc.net/problem/11522)

```en
For N line, given k, m, print k and sum of first m natural, odd, even number
```

```txt
Input:
3
1 1
2 10
3 1001

Output:
1 1 1 2
2 55 100 110
3 501501 1002001 1003002
```

## Solution

* py

  ```py
  n = int(input())
  for i in range(n):
    k, n = map(int, input().split())
    print(k, n * (n + 1) // 2, n * (n + 1) - n, n * (n + 1))
  ```
