# [BJ_16546](https://acmicpc.net/problem/16546)

```en
Find missing number from 1 - n
```

```kr
첫 줄에 선수의 수 n이 주어진다.
두번 째 줄에 n - 1 의 선수가 주어지는데 이때 1 ... n 사이에 비어있는 선수의 번호를 출력하라.
```

```txt
Input:
5
1 5 2 3

Output: 4
```

## Solution

* py

  ```py
  n = int(input())
  print(n * (n + 1) // 2 - sum(map(int, input().split())))
  ```
