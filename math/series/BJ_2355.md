# [BJ_2355](https://acmicpc.net/problem/2355)

* en

  ```en
  Given two integers A and B, find the sum of the numbers between the two integers
  ```

* tc

  ```tc
  Input: 1 10
  Output: 55
  ```

## Solution

* py

  ```py
  a, b = map(int, input().split())
  print((a + b) * (abs(a - b) + 1) // 2)
  ```
