# [BJ_4880](https://acmicpc.net/problem/4880)

Find if Arithmetic or geometric series

```txt
Input: 4 7 10
2 6 18
0 0 0
Output: AP 13
GP 54
```

## Solution

```py
while True:
  a, b, c = map(int, input().split())
  if a == b == c == 0:
    break
  if c - b == b - a:
    print(f"AP {c + (b - a)}")
  else:
    print(f"GP {c * (b // a)}")
```
