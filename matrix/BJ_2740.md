# [BJ_2740](https://acmicpc.net/problem/2740)

Multiply two N \* M, M * K matrix

```txt
Input:
3 2
1 2
3 4
5 6
2 3
-1 -2 0
0 0 3

Output:
-1 -2 6
-3 -6 12
-5 -10 18
```

## Solution

```cpp
#include <cstdio>

int i, j, k, N, M, K, A[100][100], B[100][100], r;
int main() {
  scanf("%d%d", &N, &M);
  for (; i < N; i++)
    for (j = 0; j < M; j++)
      scanf("%d", &A[i][j]);

  scanf("%*d%d", &K);
  for (i = 0; i < M; i++)
    for (j = 0; j < K; j++)
      scanf("%d", &B[i][j]);

  for (i = 0; i < N; i++, printf("\n"))
    for (j = 0; j < K; j++, printf("%d ", r))
      for (k = 0, r = 0; k < M; k++)
        r += A[i][k] * B[k][j];
}
```

```py
N, M = map(int, input().split())
A = [list(map(int, input().split())) for _ in range(N)]
M, K = map(int, input().split())
B = [list(map(int, input().split())) for _ in range(M)]
for n in range(N):
  for k in range(K):
    tmp = 0
    for m in range(M):
      tmp += A[n][m] * B[m][k]
    print(tmp, end = " ")
  print()
```
