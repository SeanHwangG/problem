# [BJ_1212](https://acmicpc.net/problem/1212)

Print oct to binary

```txt
Input: 314
Output: 11001100
```

## Solution

```go
package main

import "fmt"
import "math/big"

func main() {
  s := new(big.Int)
  fmt.Scanf("%o", s)
  fmt.Printf("%b\n", s)
}
```

```py
n = int(input(), 8)
print(bin(n)[2:])
```
