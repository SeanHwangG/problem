# [BJ_2745](https://acmicpc.net/problem/2745)

Given a, b, convert b's number of a to decimal

```txt
Input: ZZZZZ 36
Ouput: 60466175
```

## Solution

```go
package main

import (
  "fmt"
  "strconv"
)

func main() {
  n, b := "", 0
  fmt.Scan(&n, &b)
  r, _ := strconv.ParseInt(n, b, 64)
  fmt.Println(r)
}
```

```py
a, b = input().split()
print(int(a, int(b)))
```
