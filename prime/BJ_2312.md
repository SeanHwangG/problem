# [BJ_2312](https://acmicpc.net/problem/2312)

When given a positive integer N, outputs a small factorized result of this number

```txt
Input: 2
6
24

Output: 2 1
3 1
2 3
3 1
```

## Solution

```py
n = int(input())
for i in range(n):
  a = int(input())
  for i in range(2, a + 1):
  count = 0
  if a % i == 0:
    while a % i == 0:
    a = a // i
    count += 1
    print(i, count)
  elif a == 1:
    break
```
