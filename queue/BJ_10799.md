# [BJ_10799](https://acmicpc.net/problem/10799)

Given parenthesis representation of the arrangement of the iron rod and the laser
Find total number of pieces of the cut iron rod

```txt
Input: ()(((()())(())()))(())
Output: 17
```

## Solution

```py
s=input().replace('()', 'L')
a=0
num = s.count('(')
for n in s:
  if n=='(':
    a+=1
  elif n==')':
    a-=1
  elif n=='L':
    num += a

print(num)
```
