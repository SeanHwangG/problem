# [BJ_18258](https://acmicpc.net/problem/18258)

Design queue

```txt
Input: 15
push 1
push 2
front
back
size
empty
pop
pop
pop
size
empty
pop
push 3
empty
front

Output:
1
2
2
0
1
2
-1
0
1
-1
0
3
```

## Solution

```py
from collections import deque
import sys
N = int(input())
samples = deque()
for _ in range(N):
  tokens = sys.stdin.readline().split()
  if tokens[0] == 'push':
    samples.append(tokens[1])
  elif tokens[0] == 'pop':
    print(samples.popleft() if samples else -1)
  elif tokens[0] ==  'size':
    print(len(samples))
  elif tokens[0] == 'empty':
    print(0 if samples else 1)
  elif tokens[0] == 'front':
    print(samples[0] if samples else -1)
  elif tokens[0] == 'back':
    print(samples[-1] if samples else -1)
```
