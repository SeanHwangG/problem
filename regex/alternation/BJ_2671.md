# [BJ_2671](https://acmicpc.net/problem/2671)

Match following sequence (100\*1\*|01)*

```txt
Input: 10010111
Output: NOISE
```

## Solution

```go
package main

import (
  "fmt"
  "regexp"
)

func main() {
  p, s := regexp.MustCompile("^(100+1+|01)+$"), ""
  fmt.Scan(&s)
  if p.MatchString(s) {
    fmt.Println("SUBMARINE")
  } else {
    fmt.Println("NOISE")
  }
}
```

```py
import re
print(re.match(r"(100+1+|01)+$", input()) and "SUBMARINE" or "NOISE")
```
