# [BJ_2864](https://acmicpc.net/problem/2864)

If 5, 6 can be written interchangably, find min, max

```txt
Input: 1430 4862
Output:
6282 6292
```

## Solution

```py
a, b = input().split()
mx = int(a.replace('5', '6')) + int(b.replace('5', '6'))
mn = int(a.replace('6', '5')) + int(b.replace('6', '5'))
print(mn, mx)
```
