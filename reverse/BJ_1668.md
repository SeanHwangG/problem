# [BJ_1668](https://acmicpc.net/problem/1668)

Outputs number of views on left side of first line and # on right side of the second line


```txt
Input: 5
1
2
3
4
5

Output: 5
1
```

## Solution

```py
def count_visible(li):
  count, cur_mx = 0, 0
  for e in li:
    if e > cur_mx:
      count += 1
      cur_mx = e
  return count
li = [int(input()) for _ in range(int(input()))]
print(count_visible(li))
print(count_visible(reversed(li)))
```
