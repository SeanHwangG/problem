# [BJ_13610](https://acmicpc.net/problem/13610)

Print ceil(b / (b - a))

```txt
Input: 1 10
Output: 2
```

## Solution

```cpp
#include <iostream>
using namespace std;

int main() {
  int a, b;
  cin >> a >> b;
  cout << (b + b - a - 1) / (b - a);
}
```

```py
from math import ceil
a, b = map(int, input().split())
print(ceil(b / (b - a)))
```
