# [HR_text-processing-sort-2](https://www.hackerrank.com/challenges/text-processing-sort-2)

* en

  ```en
  Sort descending order numerically
  ```

* tc

  ```tc
  Input:
  1
  3
  2

  Output:
  3
  2
  1
  ```

## Solution

* sh

  ```sh
  # -n option sorts the file on the basis of the numeric field available
  sort -nr
  ```
