# [HR_text-processing-sort-4](https://www.hackerrank.com/challenges/text-processing-sort-4)

* en

  ```en
  Sort descending order numerically
  ```

* tc

  ```tc
  Input:
  9.1
  43.7
  2.2
  62.1
  2.1
  9.3

  Output:
  62.1
  43.7
  9.3
  9.1
  2.2
  2.1
  ```

## Solution

* sh

  ```sh
  sort -nr
  ```
