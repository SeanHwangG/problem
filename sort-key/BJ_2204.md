# [BJ_2204](https://acmicpc.net/problem/2204)

Input:
3
Cat
fat
bAt
4
call
ball
All
Hall
0

Output:
bAt
All

```txt
Print lexicographically (without case) first character
```

## Solution

```py
while True:
  a = int(input())
  if a == 0:
    break
  print(min([input() for _ in range(a)], key = lambda a : a.lower()))
```
