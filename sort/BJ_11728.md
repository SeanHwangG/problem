# [BJ_11728](https://acmicpc.net/problem/11728)

Print combined list in order

```txt
Input: 2 2
3 5
2 9

Output: 2 3 5 9
```

## Solution

```cpp
#include <bits/stdc++.h>

using namespace std;

int a[2000001];
int n, m;

int main() {
  cin >> n >> m;
  for (int i = 0; i < n + m; i++) {
    cin >> a[i];
  }
  sort(a, a + n + m);
  for (int i = 0; i < n + m; i++) {
    cout << a[i] << ' ';
  }
}
```
