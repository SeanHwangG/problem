# [BJ_10773](https://acmicpc.net/problem/10773)

Zero, erases most recent number
Print sum of all numbers

```txt
Input:
10
1
3
5
4
0
0
7
0
0
6

Output: 7
```

## Solution

```cpp
#include<iostream>
#include<stack>
using namespace std;

int main(){
  stack<int> st;
  int n, x;
  cin >> n;
  while(n--) {
    cin >> x;
    if(!x) st.pop();
    else st.push(x);
  }
  x = 0;
  while(st.size()){
    x += st.top();
    st.pop();
  }
  cout << x;
}
```

```py
stk = []
for _ in range(int(input())):
  num = int(input())
  if num == 0:
    stk.pop()
  else:
    stk.append(num)
print(sum(stk))
```
