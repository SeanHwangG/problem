# [BJ_15080](https://acmicpc.net/problem/15080)

Print time difference

```txt
Input: 10 : 0 : 0
11 : 0 : 0
Output: 3600
```

## Solution

```py
a, b, c = map(int, input().split(" : "))
d, e, f = map(int, input().split(" : "))
print(((d - a) * 3600 + (e - b) * 60 + f - c + 24 * 3600) % (24 * 3600))
```
