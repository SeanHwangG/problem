# [BJ_2556](https://acmicpc.net/problem/2556)

Print star in following format

```txt
Input: 2
Output:
**
**

Input: 3
Output:
***
***
***
```

## Solution

```js
t = +(require('fs').readFileSync('/dev/stdin')+'')
console.log(('*'.repeat(t) + '\n').repeat(t))
```

```py
n=int(input())
print(('*' * n + '\n') * n)
```
