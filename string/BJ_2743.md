# [BJ_2743](https://acmicpc.net/problem/2743)

Print length of string

```txt
Input: pulljima
Output: 8
```

## Solution

```c
#include <string.h>
int main() {
  char s[101];
  scanf("%s", s);
  printf("%d", strlen(s));
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
  string s;
  cin >> s;
  cout << s.size();
}
```

```py
print(len(input()))
```

```sh
read s
echo ${#s}
```
