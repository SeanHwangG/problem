# [BJ_4999](https://acmicpc.net/problem/4999)

Print "no" if second line is longer else "go"

```txt
Input: aaah
ah
Output: go
```

## Solution

```py
a = input()
b = input()
print("no" if len(b) > len(a) else "go")
```

```sh
read a
read b
if [ ${#b} -gt ${#a} ]; then
  echo 'no'
else
  echo 'go'
fi
```
