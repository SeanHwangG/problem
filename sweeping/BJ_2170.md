# [BJ_2170](https://acmicpc.net/problem/2170)

Print length of all combined lines

```txt
Input: 4
1 3
2 5
3 5
6 7

Output: 5
```

## Solution

```py
import sys
input=sys.stdin.readline
n = int(input())
a = sorted(tuple(sorted(map(int,input().split()))) for _ in range(n))
l = 0
s, t = a[0]
for u, v in a:
  if t < u:
    l += t - s
    s, t = u, v
  else:
    t = max(t, v)
print(l + t - s)
```
