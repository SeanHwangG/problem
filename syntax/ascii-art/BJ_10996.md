# [BJ_10996](https://acmicpc.net/problem/10996)

Print Star in Following format

```txt
Input: 3
Output:
* *
 *
* *
 *
* *
 *

Input: 4
* *
 * *
* *
 * *
* *
 * *
* *
 * *
```

## Solution

* py

  ```py
  n = int(input())
  for _ in range(n):
    print('* ' * (n - n // 2))
    print(' *' * (n // 2))
  ```
