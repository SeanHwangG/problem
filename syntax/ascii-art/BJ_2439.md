# [BJ_2439](https://acmicpc.net/problem/2439)

Print Star in Following format

```txt
Input: 5
Output:
    *
   **
  ***
 ****
*****
```

## Solution

```js
var c = +(require('fs').readFileSync('/dev/stdin') + '')
for (i = 1; i <= c; i++)
  console.log(' '.repeat(c - i) + '*'.repeat(i))
```

```py
n = int(input())
for i in range(1, n + 1):
  print(' ' * (n - i) + '*' * i)
```
