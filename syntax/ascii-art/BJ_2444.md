# [BJ_2444](https://acmicpc.net/problem/2444)

* en

  ```en
  Print star in following format
  ```

* tc

  ```tc
  Input: 2
  Output:
   *
  ***
   *
  ```

## Solution

* py

  ```py
  a = int(input())
  for i in range(a - 1):
    print(' ' * (a - i - 1) + ('*' * (2 * i + 1)))
  for i in range(a-1,-1,-1):
    print(' ' * (a - i - 1) + ('*' * (2 * i + 1)))
  ```
