# [BJ_2445](https://acmicpc.net/problem/2445)

```en
Print star in following format
```

```txt
Input: 5
Output:
*        *
**      **
***    ***
****  ****
**********
****  ****
***    ***
**      **
*        *
```

## Solution

* py

  ```py
  a = int(input())
  for i in range(1, a + 1):
    print('*' * i + ' ' * (2 * (a - i)) + '*' * (i))
  for k in range(1, a + 1):
    print('*' * (a - k) + ' ' * (2 * k) + '*' * (a - k))
  ```
