# [BJ_15667](https://acmicpc.net/problem/15667)

Given N print x where x^2 + x + 1 is N

```txt
Input: 21
Output: 4  # 1 + 4 + 16
```

## Solution

* js

```js
const rl = require('readline').createInterface({ input: process.stdin });
rl.on('line', (line) => {
  var n = +line - 1;
  for (k = 1; ; k++) {
    if (k * (k + 1) === n) break;
  }
  console.log(k);
})
```

* py

```py
N = int(input())-1
for x in range(1000):
  if x * (x + 1) == N:
    print(x)
    break
```
