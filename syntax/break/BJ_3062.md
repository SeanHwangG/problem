# [BJ_3062](https://acmicpc.net/problem/3062)

Check whether original and original + reversed is palindrome

```txt
Input: 4  # N
13
58
120
5056

Output:
YES
NO
YES
NO
```

## Solution

* py

```py
for _ in range(int(input())):
  n = input()
  s = str(int(n) + int(n[::-1]))
  for i in range(0, len(s)//2):
    if s[i] != s[len(s) - i - 1]:
      print('NO')
      break
  else:
    print('YES')
```
