# [BJ_5717](https://acmicpc.net/problem/5717)

```en
In each line a, b are given and input ends with 0 0
Print a + b
```

```txt
Input: 2 2  # a, b
2 3
5 5
1 1
0 0
Output: 4
5
10
2
```

## Solution

* py

  ```py
  while True:
    a, b = map(int, input().split())
    if a == b == 0:
      break
    else:
      print(a + b)
  ```
