# [BJ_16394](https://acmicpc.net/problem/16394)

* en

  ```en
  HongIk University opened in 1946
  When given a particular year, print out how old it is
  ```

* tc

  ```tc
  Input: 2018
  Output: 72
  ```

## Solution

* cpp

  ```cpp
  #import <iostream>
  int main(int n) {
    std::cin >> n;
    std::cout << n - 1946;
  }
  ```

* py

  ```py
  a = int(input())
  print(a - 1946)
  ```
