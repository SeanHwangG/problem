# [BJ_10821](https://acmicpc.net/problem/10821)

```en
Print number of comma plus 1
```

```txt
Input: 10,20,30,50,100
Output: 5
```

## Solution

* go

  ```go
  package main

  import (
    "fmt"
    "strings"
  )

  func main()  {
    var s string
    fmt.Scanf("%s", &s)
    fmt.Print(strings.Count(s, ",")+1)
  }
  ```

* js

  ```js
  console.log(require('fs').readFileSync('dev/stdin').toString().split(',').length);
  ```

* py

  ```py
  print(1 + input().count(","))
  ```
