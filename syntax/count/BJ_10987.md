# [BJ_10987](https://acmicpc.net/problem/10987)

Count number of vowels

```txt
Input: baekjoon
Output: 4
```

## Solution

* cpp

  ```cpp
  #include <cstdio>
  int cnt;
  char c;
  int main() {
    while(scanf("%c", &c) != EOF){
      if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') cnt++;
    }
    printf("%d", cnt);
    return 0;
  }
  ```

* go

  ```go
  package main

  import "fmt"

  func main() {
    var s []byte
    fmt.Scan(&s)
    r := 0
    for _, c := range s {
      switch c {
      case 'a', 'e', 'i', 'o', 'u':
        r++
      }
    }
    fmt.Println(r)
  }
  ```

* js

  ```js
  const input = require('fs').readFileSync('dev/stdin').toString();

  ```
