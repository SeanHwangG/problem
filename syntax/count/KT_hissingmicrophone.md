# [KT_hissingmicrophone](https://open.kattis.com/problems/hissingmicrophone)

* en

  ```en
  If ss is in string print 'hiss' else print 'no hiss'
  ```

* tc

  ```tc
  Input: amiss
  Output: hiss
  ```

## Solution

* py

  ```py
  s = input()
  if 'ss' in s:
    print('hiss')
  else:
    print('no hiss')
  ```
