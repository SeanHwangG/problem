# [BJ_1008](https://acmicpc.net/problem/1008)

Print a / b

```txt
Input: 1 3
Output: 0.3333333333333
```

## Solution

```cpp
#include <iostream>
using namespace std;
int main() {
  double a, b; cin >> a >> b;
  printf("%.10f", a / b);
}
```

```java
import java.util.*;

class Main{
  public static void main(String[] args) {
    Scanner a = new Scanner(System.in);
    System.out.print(a.nextInt() / a.nextDouble());
    a.close();
  }
}
```

```py
a, b = map(int, input().split())
print(a / b)
```
