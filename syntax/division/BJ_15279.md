# [BJ_15279](https://acmicpc.net/problem/15279)



```txt
Input: 2
6 5.0000
2 3.1222

Output:
60.0000 72.0000 84.0000
19.2172 38.4344 57.6517
```

## Solution

* py

  ```py
  n_test = int(input())
  for _ in range(n_test):
    beat, sec = map(float, input().split())
    print((beat - 1) / sec * 60, beat / sec * 60, (beat + 1) / sec * 60)
  ```
