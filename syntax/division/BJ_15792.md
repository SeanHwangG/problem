# [BJ_15792](https://acmicpc.net/problem/15792)

Print a / b in precision (up to 1000 digit)

```txt
Input: 1 3
Output: 0.3333333333333333333333333333
```

## Solution

* py

  ```py
  from decimal import *
  getcontext().prec = 10000
  a, b = map(Decimal, input().split())
  print(f"{a / b:.1000f}")
  ```
