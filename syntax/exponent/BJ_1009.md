# [BJ_1009](https://acmicpc.net/problem/1009)

For N line, print a ^ b % 10, (print 10 if 0)

```txt
Input: 5  # N
1 6         # a, b
3 7
6 2
10 1
9 635

Output: 1  # 1 ^ 6 % 10
7
6
10
9
```

## Solution

* py

  ```py
  for i in range(int(input())):
    a, b = map(int, input().split())
    ans = pow(a, b, 10)
    print(10 if ans == 0 else ans)
  ```
