# [BJ_11815](https://acmicpc.net/problem/11815)

* en

  ```en
  For n numbers, print 1 if square else 0 in single line
  ```

* tc

  ```tc
  Input: 2
  4 3

  Output: 1 0
  ```

## Solution

* py

  ```py
  input()

  for n in map(int, input().split()):
    if n == (int(n ** 0.5) ** 2):
      print(1, end = " ")
    else:
      print(0, end = " ")
  ```
