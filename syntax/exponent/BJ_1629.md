# [BJ_1629](https://acmicpc.net/problem/1629)

Print A ** B modular C (help(pow))

```txt
Input: 10 11 12  # A, B, C
Output: 4   # 10 ** 11 % 12
```

## Solution

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  long long A, B, C, ans = 1;
  cin >> A >> B >> C;
  while (B > 0) {
    if (B & 1) ans = ans * A % C;
    A = A * A % C;
    B >>= 1;
  }
  cout << ans;
}
```
