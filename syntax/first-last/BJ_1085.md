# [BJ_1085](https://acmicpc.net/problem/1085)

```en
Located at (x, y), find minimum length to escape from rectangle

```

```txt
Input: 6 2 10 3
Output: 1
```

## Solution

* py

  ```py
  x, y, w, h = map(int,input().split())
  print(min(x, y, w - x, h -  y))
  ```

* sh

  ```sh
  read x y w h
  a=$((x<w-x?x:w-x))
  b=$((y<h-y?y:h-y))
  echo $((a<b?a:b))
  ```
