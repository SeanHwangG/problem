# [BJ_15051](https://acmicpc.net/problem/15051)

* en

  ```en
  Print mimum of a x 4 + b x 2, a x 2 + c x 2, b x 2 + c x 4
  ```

* kr

  ```kr
  a, b, c 가 주어진다
  a * 4 + b * 2, a * 2 + c * 2, b * 2 + c * 4 중 가장 작은 수를 출력하라
  ```

* tc

  ```tc
  Input:
  10  # a
  20  # b
  30  # c

  Output: 80  (10 * 4 +20 * 2)
  ```

## Solution

* py

  ```py
  print(min(a * 4 + b * 2, a * 2 + c * 2, b * 2 + c * 4))
  ```
