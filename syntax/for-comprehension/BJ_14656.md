# [BJ_14656](https://acmicpc.net/problem/14656)

Find number of person who does not stand in the order

```txt
Input: 5  # N
3 2 5 4 1

Output: 3  # Here 3, 5, 1 are out of order

Input: 3
1 2 3

Output: 0
```

## Solution

```py
n = int(input())
a = list(map(int, input().split()))
print(sum(i + 1 != a[i] for i in range(n)))
```
