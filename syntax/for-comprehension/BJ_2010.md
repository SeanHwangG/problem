# [BJ_2010](https://acmicpc.net/problem/2010)

```en
Find number of pluggable computers given N multitabs wtih each # sockets
```

```txt
Input: 3  # N
1            # socket
1
1

Output: 1

Input: 2
5
8

Output: 12
```

## Solution

* py

  ```py
  n = int(input())
  print(sum(int(input() for _ in range(n))) - n + 1)
  ```
