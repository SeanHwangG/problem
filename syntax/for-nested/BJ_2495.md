# [BJ_2495](https://acmicpc.net/problem/2495)

Given three numbers, print longest consecutive integer in each number

```txt
Input: 12345123
17772345
22233331

Output: 1
3  # 777
4  # 3333
```

## Solution

* py

  ```py
  for _ in range(3):
    p, c, m = 0, 0, 0
    for i in input():
      c = c + 1 if p==i else 0
      m, p = c if c>m else m, i
    print(m + 1)
  ```
