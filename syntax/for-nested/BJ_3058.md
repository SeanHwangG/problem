# [BJ_3058](https://acmicpc.net/problem/3058)

Print sum of even number and minimum of even

```txt
Input: 2
1 2 3 4 5 6 7
13 78 39 42 54 93 86

Output: 12 2  # 2 + 4 + 6
260 42
```

## Solution

* py

  ```py
  N = int(input())
  for _ in range(N):
    evens = [n for n in list(map(int, input().split())) if n % 2 == 0]
    print(sum(evens), min(evens))
  ```
