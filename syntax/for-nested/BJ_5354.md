# [BJ_5354](https://acmicpc.net/problem/5354)

Given side length, print following box

```txt
Input: 3
3
5
4

Output:
###
#J#
###

#####
#JJJ#
#JJJ#
#JJJ#
#####

####
#JJ#
#JJ#
####
```

## Solution

* py

  ```py
  for _ in range(int(input())):
    a = int(input())
    print("#" * a)
    for _ in range(a - 2):
      print("#" + "J" * (a - 2) + "#")
    if a > 1:
      print("#" * a)
    print()
  ```
