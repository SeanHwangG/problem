# [BJ_9325](https://acmicpc.net/problem/9325)

Line consists of q and p. q is # of specific options Haebin wants to buy, and p is price of that option

```txt
Input: 2
10000
2
1 2000
3 400
50000
0

Output: 13200
50000
```

## Solution

* py

  ```py
  T = int(input())

  for _ in range(T):
    s = int(input())
    n = int(input())
    price = s

    for _ in range(n):
      q, p = map(int, input().split())
      price += q * p

    print(price)
  ```
