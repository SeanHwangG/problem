# [BJ_11269](https://acmicpc.net/problem/11269)

```en
How many character must be change to make the string as PERPERPER...

```

```txt
Input: SECRET
Output: 4
```

## Solution

* py

  ```py
  st = input()
  ret = 0
  for i, ch in enumerate(st):
    if i % 3 == 0 and ch != 'P':
      ret+=1
    if i % 3 == 1 and ch != 'E':
      ret+=1
    if i % 3 == 2 and ch != 'R':
      ret+=1
  print(ret)
  ```
