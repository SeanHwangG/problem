# [BJ_20674](https://acmicpc.net/problem/20674)

Given N numbers, print total sum so that the numbers are non-increassing?

```txt
Input: 3  # N
100
150  # subtract 50
200  # subtract 100

Output: 150

Input: 4  # N
10
0
9
8

Output: 17
```

## Solution

* py

  ```py
  li = list(int(input()) for _ in range(int(input())))
  mn, total = li[0], 0
  for n in li:
    if n > mn:
      total += n -mn
    mn = min(n, mn)
  print(total)
  ```
