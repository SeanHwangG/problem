# [BJ_2506](https://acmicpc.net/problem/2506)

Given N score, print total score.
If student get the questions consecutively, the score increases by 1

```txt
Input: 10  # N
1 0 1 1 1 0 0 1 1 0

Output: 10  # 1 0 1 2 3 0 0 1 2 0
```

## Solution

* py

  ```py
  input()
  S = s = 0
  for i in map(int, input().split()):
    s = (s + 1) * i
    S += s
  print(S)
  ```
