# [BJ_2576](https://acmicpc.net/problem/2576)

* en

  ```en
  Given 7 numbers, find sum of odd number and, and their minimum
  If there is no odd number, print -1
  ```

* tc

  ```tc
  Input: 12
  77
  38
  41
  53
  92
  85

  Output: 256  # 77 + 41 + 53 + 85
  41  # 41 is smallest odd number

  Input: 2
  4
  6
  8
  10
  12
  14

  Output: -1
  ```

## Solution

* go

  ```go
  package main

  import "fmt"

  func main() {
    sum := 0
    min := 100
    var num int
    for i := 0; i < 7; i++ {
      fmt.Scanln(&num)
      if (num & 1) == 0 {
        continue
      }
      sum += num
      if num < min {
        min = num
      }
    }
    if sum == 0 {
      fmt.Println("-1")
    } else {
      fmt.Println(sum)
      fmt.Println(min)
    }
  }
  ```

* py

  ```py
  sum_odd, min_odd = 0, 10000
  for i in range (7):
    a = int(input())
    if a % 2 != 0:
      sum_odd = a + sum_odd
      min_odd = min(a, min_odd)

  if min_odd == 10000:
    print(-1)
  else:
    print(sum_odd)
    print(min_odd)
  ```
