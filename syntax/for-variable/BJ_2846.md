# [BJ_2846](https://acmicpc.net/problem/2846)

```en
For N height, print length of largest uphill
```

```txt
Input: 8  # N
12 20 1 3 4 4 11 1

Output: 8  # 12 -> 20

Input: 5
1 2 1 4 6

Output: 5  # 1 -> 4 -> 6

Input: 6
10 8 8 6 4 3

Output: 0
```

## Solution

* py

  ```py
  input()
  max_climb, base, prev = 0, 999999999999, 999999999999
  for i in map(int, input().split()):
    if prev < i:
      max_climb = max(max_climb, i - base)
    else:
      base = i
    prev = i
  print(max_climb)
  ```
