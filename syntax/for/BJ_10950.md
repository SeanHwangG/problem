# [BJ_10950](https://acmicpc.net/problem/10950)

```en
For N lines, print sum of two numbers
```

```txt
Input: 2  # N
1 5
-5 2
Output: 6
-3
```

## Solution

* py

  ```py
  for _ in range(int(input())):
    a, b = map(int, input().split())
    print(a + b)
  ```

* sh

  ```sh
  read t
  for((;t--;));do
    read a b;echo $((a+b))
  done
  ```
