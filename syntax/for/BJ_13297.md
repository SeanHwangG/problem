# [BJ_13297](https://acmicpc.net/problem/13297)

```en
Print length of string for each N line
```

```kr
첫줄에 n, 다음 n 줄의 문자가 주어질 때 문자의 길이를 각각 출력하라
```

```txt
Input: 2  # N
ab
a
Output: 2
1
```

## Solution

* cpp

  ```cpp
  #include <iostream>
  #include <string>

  using namespace std;
  int main() {
    int T;
    string in;
    for (scanf("%d", &T); T--;) {
      cin >> in;
      cout << in.size() << endl;
    }
    return 0;
  }
  ```

* py

  ```py
  for _ in range(int(input())):
    print(len(input()))
  ```
