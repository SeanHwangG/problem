# [BJ_17608](https://acmicpc.net/problem/17608)

```en
Given histogram of N numbers, find number of visible bars from right
```

```txt
Input: 6  # N
6  # Hidden by 9
9  # Visible
7  # Visible
6  # Hidden by 6
4  # Hidden by 6
6  # Visible
Output: 3
```

## Solution

* py

  ```py
  a = int(input())
  li = []
  for _ in range(a):
    li.append(int(input()))

  mx, ret = 0, 0
  for n in reversed(li):
    if mx < n:
      mx = n
      ret += 1

  print(ret)
  ```
