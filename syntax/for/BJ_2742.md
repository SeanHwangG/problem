# [BJ_2742](https://acmicpc.net/problem/2742)

Print n...1 in each line

```txt
Input: 5
Output: 5
4
3
2
1
```

## Solution

* go

  ```go
  package main

  import . "fmt"

  func main() {
    var N int
    Scan(&N)
    for ; N > 0; N-- {
      Println(N)
    }
  }
  ```

* py

  ```py
  for i in range(int(input()), 0, -1):
    print(i)
  ```
