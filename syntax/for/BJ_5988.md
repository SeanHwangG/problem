# [BJ_5988](https://acmicpc.net/problem/5988)

* en

  ```en
  For N lines, print oddity
  ```

* tc

  ```tc
  Input: 2  # N
  1
  4
  Output: odd
  even
  ```

## Solution

* py

  ```py
  N = int(input())
  for _ in range(N):
    if int(input()) % 2 == 0:
      print("even")
    else:
      print("odd")
  ```
