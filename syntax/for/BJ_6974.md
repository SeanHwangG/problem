# [BJ_6974](https://acmicpc.net/problem/6974)

For N lines, print a quotient b and a remainder b in each line then one empty line

```txt
Input: 2  # N
4         # a
2         # b
5         # a
2         # b
Output: 2  # a // b
0               # a % b

2
1
```

## Solution

* py

  ```py
  n = int(input())
  for i in range(n):
    a, b = int(input()), int(input())
    print(a // b)
    print(a % b)
    print()
  ```
