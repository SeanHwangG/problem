# [BJ_10179](https://acmicpc.net/problem/10179)

```en
Print 20% discounted price up to second decimal points
```

```txt
Input:
3
100.00
59.99
20.00

Output:
$80.00
$47.99
$16.00
```

## Solution

* py

  ```py
  N = int(input())
  for _ in range(N):
    price = float(input())
    print(f"${price * 0.8:.2f}")
  ```
