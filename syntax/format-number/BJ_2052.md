# [BJ_2052](https://acmicpc.net/problem/2052)

* en

  ```en
  print 2 ** -n up to n preciesion
  ```

* tc

  ```tc
  Input: 5
  Output: 0.03125
  ```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  int main(){
    int n;
    std::cin >> n;
    printf("%.*f", n, pow(0.5,n));
  }
  ```

* py

  ```py
  n = int(input())
  print(f'{2 ** (-n):.{n}f}')
  ```
