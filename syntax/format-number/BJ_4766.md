# [BJ_4766](https://acmicpc.net/problem/4766)

```en
Given the measured temperature, print the temperature that has changed up to 2 decimal
The last number is 999
```

```txt
Input: 10.0
12.05
30.25
20
999

Output: 2.05
18.20
-10.25
```

## Solution

* py

  ```py
  last = float(input())
  while True:
    n = float(input())
    if n == 999:
      break
    print(f"{n - last:.2f}")
    last = n
  ```
