# [BJ_9316](https://acmicpc.net/problem/9316)

For N lines, Print Hello World, Judge i (1, ..., N)

```txt
Input: 3  # N

Output:
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>

  int main() {
    int i, n;
    for (std::cin >> n; i++ < n;)
      printf("Hello World, Judge %d! ", i);
  }
  ```

* go

  ```go
  package main
  import "fmt"

  func main() {
    var N int
    fmt.Scan(&N)
    for i := 0; i < N; i++ {
      fmt.Printf("Hello World, Judge %d!\n", i+1)
    }
  }
  ```

* py

  ```py
  n = int(input())
  for i in range(1, n + 1):
    print(f"Hello World, Judge {i}!")
  ```
