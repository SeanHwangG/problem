# [BJ_16017](https://acmicpc.net/problem/16017)

Print if a and d are greater than 7 and b == c ignore else answer

```txt
Input: 9  # a
6            # b
6            # c
8            # d
Output: ignore

Input: 5
6
6
8
Output: answer
```

## Solution

* py

  ```py
  a, b, c, d = int(input()), int(input()), int(input()), int(input())
  print('ignore' if a > 7 < d and b == c else 'answer')
  ```
