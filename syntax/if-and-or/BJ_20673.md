# [BJ_20673](https://acmicpc.net/problem/20673)

print White if (a is smaller or equal to 50) and (b is smaller or equal to 10)
print Red if q is greater than 30
print Yellow otherwise

```txt
Input: 50
10

Output: White
```

## Solution

* py

```py
p = int(input())
q = int(input())
if p <= 50 and q <= 10:
  print("White")
elif q > 30:
  print("Red")
else:
  print("Yellow")
```
