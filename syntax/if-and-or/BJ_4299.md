# [BJ_4299](https://acmicpc.net/problem/4299)

```en
Given sum and difference, print two numbers with larger one first, Print -1 if they are impossible
```

```kr
두 수의 합과 차가 주어질 때, 두 수를 큰 수 부터 출력하라
```

```txt
Input: 3 1
Output: 2 1
```

## Solution

* py

  ```py
  a, b = map(int, input().split())
  if a < b or (a + b) % 2 == 1:
    print(-1)
  else:
    print((a + b) // 2, (a - b) // 2)
  ```
