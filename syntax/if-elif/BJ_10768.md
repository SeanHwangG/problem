# [BJ_10768](https://acmicpc.net/problem/10768)

```en
Print if it's before Febuary 18th
```

```txt
Input: 1
7
Output: Before

Input: 8
31
Output: After

Input: 2
18
Output: Special
```

## Solution

* py

  ```py
  a, b = int(input()), int(input())
  s = 100 * a + b
  print('Before' if s < 218 else 'After' if s > 218 else 'Special')
  ```
