# [BJ_12091](https://acmicpc.net/problem/12091)

If modulo 3 is 0 Jolteon, if modulo 3 is 1 Flareon, if modulo 3 is 2 Vaporeon

```txt
Input: 3
Output: Vaporeon

Input: 25
Output: Jolteon
```

## Solution

* cpp

  ```cpp
  #include<bits/stdc++.h>
  int a;
  std::string s[3] = {"Jolteon", "Flareon", "Vaporeon"};
  int main(){
    std::cin >> a;
    std::cout << s[a % 3];
  }
  ```

* py

  ```py
  print(['Vaporeon','Jolteon','Flareon'][(1 + int(input())) % 3])
  ```
