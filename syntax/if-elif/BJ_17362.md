# [BJ_17362](https://acmicpc.net/problem/17362)

Given number, find row 

1  2   3   4   5
    8   7   6 
9  10 11 12 13
              14

```txt
Input: 7
Output: 3

Input: 13
Output 5
```

## Solution

```cpp
#include <cstdio>

int main(){
  int n;
  scanf("%d", &n);
  switch(n % 8){
    case 1:
      printf("1");
      break;
    case 2: case 0:
      printf("2");
      break;
    case 3: case 7:
      printf("3");
      break;
    case 4: case 6:
      printf("4");
      break;
    default:
      printf("5");
  }
}
```

