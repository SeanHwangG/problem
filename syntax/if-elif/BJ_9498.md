# [BJ_9498](https://acmicpc.net/problem/9498)

Find grade with following cutoff: A=90, B=80, C=70, C=60, >F)

```txt
Input: 75  # score
Output: C

Input: 100
Output: A
```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;
  int main() {
    int score;
    cin >> score;
    switch (score) {
      case 90 ... 100:
        cout << "A";
        break;
      case 80 ... 89:
        cout << "B";
        break;
      case 70 ... 79:
        cout << "C";
        break;
      case 60 ... 69:
        cout << "D";
        break;
      default:
        cout << "F";
        break;
    }
  }
  ```
