# [BJ_10162](https://acmicpc.net/problem/10162)

* en

  ```en
  With 5 min, 1 min, 10 second button, find min # button to press to exactly meet T seconds
  If it's impossible to match, print -1
  ```

* tc

  ```tc
  Input: 100
  Output: 0 1 4

  Input: 189
  Output: -1
  ```

## Solution

* py

  ```py
  n = int(input())
  if n % 10 != 0:
    print(-1)
  else:
    print(n // 300, n % 300 // 60, n % 60 // 10)
  ```
