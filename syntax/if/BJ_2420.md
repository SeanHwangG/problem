# [BJ_2420](https://acmicpc.net/problem/2420)

Print difference of two integers

```txt
Input: 2 6
Output: 4

Input: 4 -2
Output: 6
```

## Solution

* go

  ```go
  package main

  import (
    "fmt"
    "math"
  )

  func main() {
    var a, b float64
    fmt.Scanln(&a, &b)
    fmt.Println(int(math.Abs(a - b)))
  }
  ```

* php

  ```php
  <?
  fscanf (STDIN,"%d %d", $a, $b);
  echo abs ($a - $b);
  ?>
  ```

* py

  ```py
  a, b = map(int, input().split())
  print(abs(a - b))
  ```
