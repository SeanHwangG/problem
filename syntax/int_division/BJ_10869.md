# [BJ_10869](https://acmicpc.net/problem/10869)

* en

  ```en
  Print add, subtract, multiply, division, remainer in each line
  ```

* tc

  ```tc
  Input: 7 3
  Output: 10
  4
  21
  2
  1
  ```

## Solution

* cpp

  ```cpp
  #import<iostream>
  int main() {
    int a, b;
    std::cin >> a >> b;
    printf("%d\n%d\n%d\n%d\n%d", a + b, a - b, a * b, a / b, a % b);
  }
  ```

* py

  ```py
  a, b = map(int, input().split())
  print(a + b, a - b, a * b, a // b, a % b, sep='\n')
  ```

* sh

  ```sh
  read A B
  echo $(($A+$B))
  echo $(($A-$B))
  echo $(($A*$B))
  echo $(($A/$B))
  echo $(($A%$B))
  ```
