# [BJ_15727](https://acmicpc.net/problem/15727)

```en
Can travel a distance of 1 to 5 per minute
Given distance from voice actor's house to house, print how many minutes you can find him
```

```txt
Input: 12
Output: 3
```

## Solution

* sh

  ```sh
  read a
  echo $(((a + 4)/5))
  ```

* py

  ```py
  a = int(input())
  print((a + 4) // 5)
  ```
