# [BJ_18301](https://acmicpc.net/problem/18301)

* en

  ```en
  (Multiple of a + 1 and b + 1 divdied by c + 1) minus 1
  ```

* kr

  ```kr
  a + 1 와 b + 1 의 곱을 c + 1로 나눈 몫에 1를 뺀 수를 출력하라
  ```

* tc

  ```tc
  Input: 15 18 11
  Output: 24
  ```

## Solution

* py

  ```py
  a, b, c = map(int, input().split())
  print((a + 1) * (b + 1) // (c + 1) - 1)
  ```
