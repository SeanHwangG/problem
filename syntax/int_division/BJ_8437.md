# [BJ_8437](https://acmicpc.net/problem/8437)

* en

  ```en
  Given a, b print average and difference divided by 2
  ```

* kr

  ```kr
  a, b 가 한 줄로 주어질 때
  a + b를 2로 나눈 몫과, a - b를 2로 나눈 몫을 각각 출력하라
  ```

* tc

  ```tc
  Input: 5 2
  Output: 3  # (5 + 2) / 2
  1 # (5 - 2) / 2

  Input: 2 10
  Output: 6
  4
  ```

## Solution

* py

  ```py
  a = int(input())
  b = int(input())
  print((a + b) // 2)
  print((a - b) // 2)
  ```
