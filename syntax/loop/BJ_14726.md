# [BJ_14726](https://acmicpc.net/problem/14726)

```en
For odd digit, multiply by 2 (if it exceeds 10, sum digits)
Check sum is divisible by 10
```

```txt
Input:
3
2720992711828767
3444063910462763
6011733895106094

Output:
T
F
T
```

## Solution

* py

  ```py
  for i in range(int(input())):
    s = 0
    n = input()
    for i in range(1, 17):
      m = int(n[i-1])*(i % 2 + 1)
      s += m // 10 + m % 10
    print('FT'[s % 10 == 0])
  ```
