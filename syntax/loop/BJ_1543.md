# [BJ_1543](https://acmicpc.net/problem/1543)

* en

  ```en
  Count how many times second sentence appear in first sentence
  ```

* tc

  ```tc
  Input:
  ababababa
  aba

  Output: 2
  ```

## Solution

* py

  ```py
  doc = input()
  word = input()
  count = 0
  i = 0
  while i <= len(doc) - len(word):
    if doc[i:i + len(word)] == word:
      count += 1
      i += len(word)
    else:
      i += 1
  print(count)
  ```
