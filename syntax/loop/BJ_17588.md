# [BJ_17588](https://acmicpc.net/problem/17588)



```txt
Input:
9
2
4
5
7
8
9
10
11
13

Output:
1
3
6
12
```

## Solution

* py

```py
N = int(input())
li = list(int(input()) for _ in range(N))

if len(li) == li[-1]:
  print("good job")
for n in range(1, li[-1] + 1):
  if n not in li:
    print(n)
```
