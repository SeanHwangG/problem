# [BJ_5789](https://acmicpc.net/problem/5789)

Print 'Do-it' if middle two are same else 'Do-it-Not'

```txt
Input:
3
00100010
01010101
100001

Output:
Do-it
Do-it-Not
Do-it
```

## Solution

* py

  ```py
  for _ in[0]*int(input()):
    s = input()
    a = len(s) // 2
    print('Do-it'+'-Not'*(s[a]!=s[a-1]))
  ```
