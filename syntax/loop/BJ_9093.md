# [BJ_9093](https://acmicpc.net/problem/9093)

Print each line in reverse

```txt
Input:
2
I am happy today
We want to win the first prize

Output:
I ma yppah yadot
eW tnaw ot niw eht tsrif ezirp
```

## Solution

* py

  ```py
  for _ in range(int(input())):
    for st in input().split():
      print(st[::-1], end=' ')
  ```
