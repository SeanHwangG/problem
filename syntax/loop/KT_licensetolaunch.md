# [KT_licensetolaunch](https://open.kattis.com/problems/licensetolaunch)

* en

  ```en
  Given N ints, print min index
  ```

* kr

  ```kr
  N개의 숫자가 주어질 때 최소인 index를 출력하라
  ```

* tc

  ```tc
  Input:
  5
  3 4 1 7 2

  Output: 2
  ```

## Solution

* py

  ```py
  input()
  s = input()
  print(s.index(min(s)))
  ```
