# [BJ_10156](https://acmicpc.net/problem/10156)

```en
Given the price of snack (K),  number of snacks (N), Money owned (M)
Print extra amount of money you need to buy all snacks
```

```txt
Input: 300 4 1000
Output: 200
```

## Solution

* py

  ```py
  a, b, c = map(int, input().split())
  print(max(0, a * b - c))
  ```
