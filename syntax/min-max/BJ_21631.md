# [BJ_21631](https://acmicpc.net/problem/21631)

* en

  ```en
  Print smaller number between between a + 1, b
  ```

* kr

  ```kr
  a + 1 과 b 중 작은 수를 출력하라
  ```

* tc

  ```tc
  Input: 1 2
  Output: 2
  ```

## Solution

* py

  ```py
  a, b = map(int,input().split())
  print(min(a + 1, b))
  ```
