# [BJ_10941](https://acmicpc.net/problem/10941)

```en
Decode given BASE16 encoded string
```

```txt
Input: 4261656B6A6F6F6E
Output: Baekjoon
```

## Solution

* cpp

  ```cpp
  #include <cstdio>
  int main(){
    int n;
    while (scanf("%2X",&n)!=EOF)
      printf("%c",n);
  }
  ```

* py

  ```py
  import base64
  print(base64.b16decode(input()).decode())
  ```
