# [BJ_11719](https://acmicpc.net/problem/11719)

Print all input

```txt
Input:
    Hello

Baekjoon
   Online Judge

Output:
    Hello

Baekjoon
   Online Judge
```

## Solution

* cpp

  ```cpp
  #include <iostream>
  int main() {
    char c;
    while ((c = std::cin.get()) != -1) {
      std::cout << c;
    }
  }
  ```

* py

  ```py
  import sys
  print(sys.stdin.read())
  ```
