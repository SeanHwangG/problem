# [BJ_11931](https://acmicpc.net/problem/11931)

* en

  ```en
  Print list in reversed order
  ```

* tc

  ```tc
  Input:
  5
  1
  2
  3
  4
  5

  Output:
  5
  4
  3
  2
  1
  ```

## Solution

* py

  ```py
  import sys
  input = sys.stdin.readline
  n = int(input())
  li = [int(input()) for _ in range(n)]
  for e in sorted(li, reverse=True):
    print(e)
  ```
