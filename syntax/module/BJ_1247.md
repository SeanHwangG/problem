# [BJ_1247](https://acmicpc.net/problem/1247)

* en

  ```en
  Given N integers, find the sign of these summed integers
  ```

## Solution

* cpp

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
    for (int c = 0; c < 3; c++) {
      int n; cin >> n;
      __int128 s = 0;
      for (int i = 0; i < n; i++) {
        long long a; cin >> a; s += a;
      }
      if (s > 0) cout << '+' << '\n';
      else if (s < 0) cout << '-' << '\n';
      else cout << 0 << '\n';
    }
  }
  ```

* py

  ```py
  import sys
  for _ in range(3):
    N = int(input())
    li = [int(sys.stdin.readline()) for _ in range(N)]
    total = sum(li)
    if total > 0:
      print('+')
    elif total < 0:
      print('-')
    else:
      print(0)
  ```
