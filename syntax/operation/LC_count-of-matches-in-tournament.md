# [LC_count-of-matches-in-tournament](https://leetcode.com/problems/count-of-matches-in-tournament)

```en
In tournament, find minimum number of matches until winner
```

```txt
Input: n = 7
Output: 6
```

## Solution

* py

  ```py
  def numberOfMatches(self, n: int) -> int:
    return n - 1
  ```
