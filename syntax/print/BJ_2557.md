# [BJ_2557](https://acmicpc.net/problem/2557)

print the following

```txt
Output: Hello World!
```

## Solution

* asm

  ```asm
  section .data
    hello: db "Hello World!"
  section .text
    global main
  main:
    mov   rax, 1
    mov   rdi, 1
    mov   rsi, hello
    mov   rdx, 12
    syscall
  ```

* go

  ```go
  package main

  import "fmt"

  func main() {
    fmt.Println("Hello World!")
  }
  ```

* py

  ```py
  print("Hello World!")
  ```

* sh

  ```sh
  echo Hello World!
  ```

* sv

  ```sv
  module test;
    initial begin
      $display("Hello World!");
    end
  endmodule
  ```
