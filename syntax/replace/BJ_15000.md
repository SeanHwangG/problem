# [BJ_15000](https://acmicpc.net/problem/15000)

Print given string in uppercase

```txt
Input: alert
Output: ALERT
```

## Solution

* cpp

  ```cpp
  #include <stdio.h>
  int main(){
    char x[1000001];
    scanf("%s",x);
    for (int i = 0; x[i]; i++)
      printf("%c", x[i] - 32);
  }
  ```

* py

  ```py
  print(input().upper())
  ```
