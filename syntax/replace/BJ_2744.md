# [BJ_2744](https://acmicpc.net/problem/2744)

```en
Swap case
```

```txt
Input: Hi
Output: hI
```

## Solution

* cpp

  ```cpp
  #include<cstdio>
  char w[101];

  int main() {
    scanf("%s", w);
    for (int i = 0; w[i]; ++i) {
      printf("%c", w[i] ^ 32);
    }
  }
  ```

* py

  ```py
  st = input()
  for ch in st:
    if ch.islower():
      print(ch.upper(), end='')
    else:
      print(ch.lower(), end='')
  ```
