# [BJ_2810](https://acmicpc.net/problem/2810)

```en
Determines maximum # people that can fit a cup into the cupholder
```

```txt
Input: 9 SLLLLSSLL
Output: 7
```

## Solution

* py

  ```py
  input()
  s = input()
  if 'L' not in s:
    print(len(s))
  else:
    print(len(s.replace("LL", "S")) + 1)
  ```
