# [BJ_2941](https://acmicpc.net/problem/2941)

Print length of string given followings sequences are considered as one character
c-, c=, d-, dz=, lj, nj, s=, z=

```txt
Input: ljes=njak
Output: 6  # Total 6 character (lj, e, s=, nj, a, k)
Input: ddz=z=
Output: 3
```

## Solution

* py

```py
import re
print(len(re.sub('dz=|[ln]j|\w\W','Z',input())))
```

* sh

```sh
read a
a="${a//c=/.}"
a="${a//c-/.}"
a="${a//dz=/.}"
a="${a//d-/.}"
a="${a//lj/.}"
a="${a//nj/.}"
a="${a//s=/.}"
a="${a//z=/.}"
echo ${#a}
```
