# [BJ_5524](https://acmicpc.net/problem/5524)

* en

  ```en
  Print N names in lower case
  ```

* tc

  ```tc
  Input: 3  # N
  WatanabE
  ITO
  YamaMoto
  Output: watanabe
  ito
  yamamoto
  ```

## Solution

* cpp

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;
  int main() {
    int n;
    cin >> n;
    while (n--) {
      string a;
      cin >> a;
      transform(a.begin(), a.end(), a.begin(), ::tolower);
      cout << a << "\n";
    }
  }
  ```

* py

  ```py
  for _ in range(int(input())):
    print(input().lower())
  ```
