# [LC_reverse-string](https://leetcode.com/problems/reverse-string)

* en

  ```en
  Reverse array of characters
  ```

* tc

  ```tc
  Input: s = ["h","e","l","l","o"]
  Output:
  ["o","l","l","e","h"]
  ```

## Solution

* py

  ```py
  def reverseWords(self, s: str) -> str:
    return s[::-1]
  ```
