# [LC_reverse-words-in-a-string](https://leetcode.com/problems/reverse-words-in-a-string)

* en

  ```en
  Reverse word
  ```

* tc

  ```tc
  Input: s = "the sky is blue"
  Output: "blue is sky the"
  ```

## Solution

* py

  ```py
  def reverseWords(self, s: str) -> str:
    return s[::-1]
  ```
