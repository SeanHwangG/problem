# [BJ_13610](https://acmicpc.net/problem/13610)

```en
Print ceil(b / (b - a))
```

```kr
b / (b - a) 를 출력하라. 나누어 떨어지지 않으면 올림하라
```

```txt
Input: 1 10
Output: 2
```

## Solution

* cpp

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
    int a, b;
    cin >> a >> b;
    cout << (b + b - a - 1) / (b - a);
  }
  ```

* py

  ```py
  from math import ceil
  a, b = map(int, input().split())
  print(ceil(b / (b - a)))
  ```
