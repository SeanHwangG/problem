# [BJ_1100](https://acmicpc.net/problem/1100)

* en

  ```en
  Given the status of the chessboard, print out how many horses are on the white compartmen

  ```

* tc

  ```tc
  Input:
  .F.F...F
  F...F.F.
  ...F.F.F
  F.F...F.
  .F...F..
  F...F.F.
  .F.F.F.F
  ..FF..F.

  Output: 1
  ```

## Solution

* java

  ```java
  import java.util.*;
  public class Main {
    public static void main(String[] args) {
      Scanner scan = new Scanner(System.in);
      int count = 0;
      for (int i = 0; i < 8; i++) {
        String line = scan.nextLine();
        for(int j = 0; j < 8; j++)
          if(line.charAt(j) == 'F' && (j + i) % 2 == 0) count++;
      }
      System.out.print(count);
      scan.close();
    }
  }
  ```

* py

  ```py
  r = ''
  for _ in range(8):
    r += input() + '0'
  print(r[::2].count('F'))
  ```

* sh

  ```sh
  col=( 0 2 4 6 )
  tot=0
  for ((i=0; i<8; i++)){
    read board
    row=$(($i%2))
    for c in ${col[@]}; do
      if [ ${board:$(($row+$c)):1}=F ] ; then
        let tot+=1
      fi
    done
  }
  echo $tot
  ```
