# [BJ_1292](https://acmicpc.net/problem/1292)

* en

  ```en
  Given sequence 122333.. and A, B, find the sum of the digits A to B in the sequence
  ```

* tc

  ```tc
  Input: 3 7
  Output: 15  # 1 2 (2 3 3 3 4)
  ```

## Solution

* py

  ```py
  number_list = []
  for i in range(1, 46):
    number_list += [i] * i

  A, B = map(int, input().split())
  print(sum(number_list[A-1:B]))
  ```

* sh

  ```sh
  arr=(0)
  for i in {1..45}; do
    for (( j=0; j<i; ++j )); do
      arr+=($i)
    done
  done

  read a b
  sum=0
  for (( i=a; i<=b; ++i )); do
  ```
