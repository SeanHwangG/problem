# [BJ_18406](https://acmicpc.net/problem/18406)

* en

  ```en
  If sum of left and right are same print LUCKY else print READY
  ```

* tc

  ```tc
  Input: 123402
  Output: LUCKY
  ```

## Solution

* py

  ```py
  tmp=input()
  if sum(map(int, str(tmp[:len(tmp)//2]))) == sum(map(int, str(tmp[len(tmp)//2:]))):
    print("LUCKY")
  else:
    print("READY")
  ```
