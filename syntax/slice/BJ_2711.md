# [BJ_2711](https://acmicpc.net/problem/2711)

```en
Remove nth character from string
```

```txt
Input:
4
4 MISSPELL
1 PROGRAMMING
7 CONTEST
3 BALLOON

Output:
MISPELL
ROGRAMMING
CONTES
BALOON
```

## Solution

* py

  ```py
  for _ in range(int(input())):
    a, b = input().split()
    print(b[:int(a) - 1]+b[int(a):])
  ```
