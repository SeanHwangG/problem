# [BJ_1427](https://acmicpc.net/problem/1427)

Print number by digits in descending order

```txt
Input: 2143
Output: 4321
```

## Solution

* go

  ```go
  package main

  import (
    "fmt"
    "sort"
  )

  func main() {
    var n int
    var arr []int

    fmt.Scan(&n)

    for i := 0; n > 0; i++ {
      arr = append(arr, n%10)
      n = n / 10
    }

    sort.Sort(sort.Reverse(sort.IntSlice(arr)))
    for i := 0; i < len(arr); i++ {
      fmt.Print(arr[i])
    }
  }
  ```

* py

  ```py
  a = input()
  print("".join(sorted(a, reverse=True)))
  ```
