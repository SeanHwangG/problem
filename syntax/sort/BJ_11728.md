# [BJ_11728](https://acmicpc.net/problem/11728)

Given two list with N and M number of elements each, print merged list in order

```txt
Input: 2 2  # N, M
3 5
2 9

Output: 2 3 5 9

Input: 4 3
2 3 5 9
1 4 7
```

## Solution

```cpp
#include <bits/stdc++.h>

using namespace std;

int a[2000001];
int n, m;

int main() {
  cin >> n >> m;
  for (int i = 0; i < n + m; i++) {
    cin >> a[i];
  }
  sort(a, a + n + m);
  for (int i = 0; i < n + m; i++) {
    cout << a[i] << ' ';
  }
}
```
