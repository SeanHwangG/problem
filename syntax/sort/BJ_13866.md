# [BJ_13866](https://acmicpc.net/problem/13866)

Given four numbers, split them into two pairs so that their difference is minimum

```txt
Input: 4 7 10 20
Output: 7  # (20 + 4) - (7 + 10)

Input: 1 2 3 4
Output: 0  # (1 + 4) - (2 + 3)
```

## Solution

```py
a, b, c, d = sorted(map(int, input().split()))
print(abs(d + a - (b + c)))
```
