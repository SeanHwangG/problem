# [BJ_2751](https://acmicpc.net/problem/2751)

Print list in ascending order

```txt
Input: 5  # N (1 ≤ N ≤ 1,000,000)
5
2
3
4
1

Output: 1
2
3
4
5
```

## Solution

```php
<?php
  fscanf(STDIN,"%d",$n);
  for ($i=0;$i<$n;$i++)
  {
    fscanf(STDIN,"%d",$arr[$i]);
  } sort($arr);
  for ($i=0;$i<$n;$i++)
    echo "$arr[$i] ";
?>
```

```py
import sys
input = sys.stdin.readline
li = [int(input()) for _ in range(int(input()))]
for n in sorted(li):
  print(n)
```
