# [BJ_2997](https://acmicpc.net/problem/2997)

When four are aligned, difference between adjacent pairs is constant
When three of four numbers are given, print fourth number

```txt
Input: 4 6 8
Output: 10
```

## Solution

* py

  ```py
  a, b, c = sorted(map(int,input().split()))
  if b - a == c - b:
    print(2 * c - b)
  elif b - a < c - b:
    print((b + c) // 2)
  else:
    print((a + b) // 2)
  ```
