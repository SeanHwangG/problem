# [BJ_10824](https://acmicpc.net/problem/10824)

```en
Print (a concat b) + (c concat d)
```

```txt
Input: 10 20 30 40
Output: 4060  # 1020 + 3040
```

## Solution

* js

  ```js
  var t = (require('fs').readFileSync('/dev/stdin')+'').split(' ');
  console.log(+(t[0] + t[1])+ +(t[2] + t[3]));
  ```

* py

  ```py
  a, b, c, d = input().split()
  print(int(a + b) + int(c + d))
  ```
