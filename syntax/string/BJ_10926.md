# [BJ_10926](https://acmicpc.net/problem/10926)

Print input with ??!

```txt
Input: joonas
Output: joonas??!
```

## Solution

* go

  ```go
  package main

  import "fmt"

  func main() {
    var s string
    fmt.Scan(&s)
    fmt.Println(s+"??!")
  }
  ```

* py

  ```py
  print(input() + "??!")
  ```
