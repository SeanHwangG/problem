# [BJ_1402](https://acmicpc.net/problem/1402)

* en

  ```en
  Print yes for n times
  ```

* kr

  ```kr
  들어온 숫자만 큼 n을 출력하라
  ```

* tc

  ```tc
  Input: 3
  Output: yes
  yes
  yes
  ```

## Solution

* go

  ```go
  package main

  import "fmt"
  import "strings"

  func main() {
    var t int
    fmt.Scan(&t)
    fmt.Println(strings.Repeat("yes\n", t))
  }
  ```

* py

  ```py
  for _ in range(int(input())):
    print('yes')
  ```
