# [BJ_14581](https://acmicpc.net/problem/14581)

Print in following format

```txt
Input: appa
Output:
:fan::fan::fan:
:fan::appa::fan:
:fan::fan::fan:
```

## Solution

* go

  ```go
  package main

  import "fmt"

  func main() {
    var s string
    fmt.Scan(&s)
    fmt.Print(`:fan::fan::fan:
  :fan::`, s,`::fan:
  :fan::fan::fan:`)
  }
  ```

* py

  ```py
  print(f""":fan::fan::fan:")
  :fan::{input()}::fan:
  :fan::fan::fan:""")
  ```
