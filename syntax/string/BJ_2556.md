* tc

  ```tc
  Input: 2
  Output:
  **
  **

  Input: 3
  Output:
  ***
  ***
  ***
  ```

## Solution

* js

  ```js
  t = +(require('fs').readFileSync('/dev/stdin')+'')
  console.log(('*'.repeat(t) + '\n').repeat(t))
  ```

* py

  ```py
  n=int(input())
  print(('*' * n + '\n') * n)
  ```
