# [BJ_10262](https://acmicpc.net/problem/10262)

* en

  ```en
  Print Gunnar if sum of first list is bigger tie if same else Emma
  ```

* tc

  ```tc
  Input:
  1 4 1 4
  1 6 1 6

  Output:
  Emma
  ```

## Solution

* py

  ```py
  tmp1=list(map(int,input().split()))
  tmp2=list(map(int,input().split()))
  a = sum(tmp1)
  b = sum(tmp2)
  if a > b: print("Gunnar")
  elif a == b: print("Tie")
  else: print("Emma")
  ```
