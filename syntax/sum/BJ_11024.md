# [BJ_11024](https://acmicpc.net/problem/11024)

* en

  ```en
  Print sum of lists for n test
  ```

* tc

  ```tc
  Input:
  2
  1 5 3
  1 7

  Output:
  9
  8
  ```

## Solution

* py

  ```py
  N = int(input())
  for _ in range(N):
    n = list(map(int, input().split()))
    print(sum(n))
  ```
