# [BJ_20332](https://acmicpc.net/problem/20332)

```en
Print if sum of numbers in second line are divisible by 3
```

```kr
첫 줄에 n 이 주어지고
두번 째 줄에 n개의 숫자가 주어질 때, 합이 3으로 나누어 떨어지면 yes 아니면 no를 출력하라.
```

```txt
Input:
2
10 3

Output: no
```

## Solution

* py

  ```py
  input()
  print("yes" if sum(map(int, input().split())) % 3 == 0 else "no")
  ```
