# [BJ_2490](https://acmicpc.net/problem/2490)

* en

  ```en
  Prints A for do, B for gae, C for gul, D for yut, and E for mo
  ```

* tc

  ```tc
  Input:
  0 1 0 1
  1 1 1 0
  0 0 1 1

  Output:
  B
  A
  B
  ```

## Solution

* py

  ```py
  for i in range(3):
    s = sum(int, input().split())
    if s == 0:
      print('D')
    elif s == 1:
      print('C')
    elif s == 2:
      print('B')
    elif s == 3:
      print('A')
    else:
      print('E')
  ```
