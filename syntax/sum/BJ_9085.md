# [BJ_9085](https://acmicpc.net/problem/9085)

* en

  ```en
  Print sum of list in each test case
  ```

* tc

  ```tc
  Input:
  2
  1 4 2
  1 0

  Output:
  7
  1
  ```

## Solution

* py

  ```py
  for _ in range(int(input())):
    m = int(input())
    print(sum(map(int, input().split())))
  ```
