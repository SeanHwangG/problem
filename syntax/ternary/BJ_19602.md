# [BJ_19602](https://acmicpc.net/problem/19602)

```en
If a + 2 x b + 3 x c < 10 print sad else print happy
```

```kr
a + 2 * b + 3 * c < 10 이면 sad 아니면 happy 를 출력하라
```

```txt
Input: 3
1
0
Output: sad
```

## Solution

* py

  ```py
  a = int(input())
  b = int(input())
  c = int(input())
  print('sad' if a + 2 * b + 3 * c < 10 else 'happy')
  ```
