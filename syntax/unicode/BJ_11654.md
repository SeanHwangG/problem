# [BJ_11654](https://acmicpc.net/problem/11654)

Print ascii of character

```txt
Input: C
Output: 67
```

## Solution

* cpp

  ```cpp
  #include<bits/stdc++.h>
  using namespace std;

  int main() {
    char a;
    cin >> a;
    cout << (int)a;
    return 0;
  }
  ```

* py

  ```py
  print(ord(input()))
  ```

* sh

  ```sh
  read n
  printf "%d" \'$n
  ```

* sv

  ```sv
  module main;
    initial begin
      integer code;
      logic[7:0] c;
      code = $fscanf(32'h8000_0000, "%c", c);
      $display("%0d", c);
      $finish;
    end
  endmodule
  ```
