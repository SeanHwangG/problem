# [BJ_5598](https://acmicpc.net/problem/5598)

Given string, shifting three character (A->X, B->Y, C->Z, D->A, E->B, ...)

```txt
Input: MRL
Output: JOI
```

## Solution

```go
package main

import "fmt"

func main() {
  m := map[byte]byte{'A': 'X', 'B': 'Y', 'C': 'Z'}
  for i := byte('D'); i <= byte('Z'); i++ {
    m[i] = i - 3
  }
  var s []byte
  fmt.Scan(&s)
  for i, c := range s {
    s[i] = m[c]
  }
  fmt.Printf("%s\n", s)
}
```

```py

for ch in st:
```
