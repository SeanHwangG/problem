# [BJ_6321](https://acmicpc.net/problem/6321)

For N lines, shift alphabet to right one character (A->B, B->C, ..., Z -> A)

```txt
Input: 2
HAL
SWERC

Output:
String #1
IBM  # H->I, A->B, L->M
String #2
TXFSD
```

## Solution

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int n;
  cin >> n;
  for (int i = 1; i <= n; i++){
    string s;
    cin >> s;
    for (auto& c: s)
      if (c++ == 'Z')
        c = 'A';
    printf("String #%d\n%c\n\n", i, s);
  }
}
```

```py
for c in range(1,int(input())+1):
```
