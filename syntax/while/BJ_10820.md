# [BJ_10820](https://acmicpc.net/problem/10820)

Find # lowercase letters, uppercase letters, numbers, and spaces in the string


```txt
Input:
This is String
SPACE    1    SPACE
 S a M p L e I n P u T
0L1A2S3T4L5I6N7E8

Output:
10 2 0 2
0 10 1 8
5 6 0 16
0 8 9 0
```

## Solution

* go

```go
package main

import (
  "bufio"
  "fmt"
  "os"
)

var in = bufio.NewScanner(os.Stdin)

func main() {
  for in.Scan() {
    var a, b, c, d int
    for _, s := range in.Bytes() {
      switch {
      case 'a' <= s && s <= 'z':
        a++
      case 'A' <= s && s <= 'Z':
        b++
      case '0' <= s && s <= '9':
        c++
      case s == ' ':
        d++
      }
    }
    fmt.Println(a, b, c, d)
  }
}
```

* py

```py
import sys
while True:
  line = sys.stdin.readline().rstrip('\n')
  up, lo, sp, nu = 0, 0, 0, 0
  if not line:
    break
  for l in line:
    if l.isupper():
      up += 1
    elif l.islower():
      lo += 1
    elif l.isdigit():
      nu += 1
    elif l.isspace():
      sp += 1
  print(f"{lo} {up} {nu} {sp}")
```
