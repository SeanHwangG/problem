# [BJ_11034](https://acmicpc.net/problem/11034)

```en
Print maximum possible jump from middle person
```

```txt
Input: 2 3 5
3 5 9
Output:
1
3
```

## Solution

* py

  ```py
  while True:
    try:
      a, b, c = map(int, input().split())
      print(max(b - a, c - b) - 1)
    except:
      break
  ```
