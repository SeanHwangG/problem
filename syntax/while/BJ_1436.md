# [BJ_1436](https://acmicpc.net/problem/1436)

* en

  ```en
  Find nth smallest number with 666 in
  ```

* tc

  ```tc
  Input: 2
  Output:
  1666
  ```

## Solution

* py

  ```py
  name = 666
  cnt=0
  while True:
    if "666" in str(name):
      cnt+=1
      if cnt == n :
        print(name)
        break
    name+=1
  ```
