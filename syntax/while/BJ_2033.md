```kr
정수 N이 주어져 있을 때 이 수가 10보다 크면 일의 자리에서 반올림을 하고, 이 결과가 100보다 크면 다시 10의 자리에서 반올림... (이하 생략)
이러한 연산을 한 결과를 출력
```

```txt
Input: 15
Output:
20
```

## Solution

* py

  ```py
  N = int(input())
  d = 10
  while N > d:
    if N % d >= d // 2:
      N += d
    N -= (N % d)
    d *= 10
  print(N)
  ```
