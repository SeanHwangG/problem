# [BJ_15680](https://acmicpc.net/problem/15680)

Print YONSEI if 0 or "Leading the Way to the Future"

```txt
Input: 0
Output: YONSEI

Input: 1
Output: Leading the Way to the Future
```

## Solution

```py
a = int(input())
print('YONSEI' if input() == '0' else 'Leading the Way to the Future')
```
