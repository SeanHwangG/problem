# [BJ_11654](https://acmicpc.net/problem/11654)

Print ascii of character

```txt
Input: C
Output: 67
```

## Solution

```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
  char a;
  cin >> a;
  cout << (int)a;
  return 0;
}
```
