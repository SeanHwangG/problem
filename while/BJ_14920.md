# [BJ_14920](https://acmicpc.net/problem/14920)

C(n+1) = C(n)/2 (when C(n) is even)
3*C(n)+1 (C(n) is odd)
For the given input C (1), output n where C(n) first becomes 1

```txt
Input: 26
Output:
11
```

## Solution

```py
cn = int(input())
count = 1
while cn != 1:
  count += 1
  if cn % 2 == 0:
    cn = cn // 2
  else:
    cn = 3 *cn + 1
print(count)
```
